# Applications mobiles et de bureau OneUptime

OneUptime propose des applications mobiles et de bureau similaires aux applications natives grâce à la technologie Progressive Web App (PWA). Ces applications offrent une expérience transparente sur toutes les principales plateformes sans nécessiter de téléchargements depuis les boutiques d'applications.

## Qu'est-ce qu'une Progressive Web App (PWA) ?

Une Progressive Web App est une application web qui utilise les capacités web modernes pour offrir une expérience similaire à celle d'une application native aux utilisateurs.

## Prise en main

1. Visitez votre instance OneUptime dans un navigateur pris en charge
2. Recherchez l'invite « Installer » ou « Ajouter à l'écran d'accueil »
3. Suivez les étapes d'installation spécifiques à votre plateforme
4. Lancez l'application depuis le tiroir d'applications ou le bureau de votre appareil

## Dépannage

### Problèmes courants

**L'option d'installation n'apparaît pas :**
- Assurez-vous d'utiliser un navigateur pris en charge
- Vérifiez que votre instance OneUptime est servie via HTTPS
- Essayez d'actualiser la page ou de vider le cache du navigateur

**Les notifications push ne fonctionnent pas :**
- Accordez les permissions de notification lorsque vous y êtes invité
- Vérifiez les paramètres de notification du navigateur
- Assurez-vous que l'instance OneUptime a les notifications configurées
