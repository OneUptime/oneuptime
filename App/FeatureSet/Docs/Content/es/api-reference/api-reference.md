# Referencia de la API

OneUptime proporciona una API REST completa que te permite integrar funcionalidades de monitoreo, gestión de incidentes y páginas de estado en tus aplicaciones y flujos de trabajo. **Todo lo que puedes hacer en el panel de OneUptime también puede hacerse a través de la API, lo que permite la automatización y las integraciones personalizadas.**

### Primeros pasos

Nuestra API está organizada en torno a los principios REST y utiliza códigos de respuesta HTTP estándar, autenticación y verbos. Todos los puntos de conexión de la API devuelven respuestas JSON.


### Autenticación

Todas las solicitudes de API requieren autenticación mediante claves de API. Puedes generar claves de API desde tu panel de OneUptime en Configuración > Claves de API.
