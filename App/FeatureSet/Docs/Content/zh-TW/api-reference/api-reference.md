# API 參考

OneUptime 提供全面的 REST API，讓您能將監控、事件管理與狀態頁面功能整合到您的應用程式與工作流程中。**凡是您能在 OneUptime 儀表板上完成的操作，皆可透過 API 達成，進而實現自動化與自訂整合。**

### 開始使用

我們的 API 圍繞 REST 原則設計，並使用標準的 HTTP 回應碼、驗證機制與動詞。所有 API 端點皆回傳 JSON 回應。


### 驗證

所有 API 請求皆需使用 API 金鑰進行驗證。您可以從 OneUptime 儀表板的 Settings > API Keys 中產生 API 金鑰。

### API 參考

請點此查看 OneUptime 的 API 參考 ➡️ [OneUptime API Reference](/reference)。此 API 參考提供多種語言版本 — 系統會依您的瀏覽器自動偵測偏好的語言，您也可隨時透過頂部導覽列中的選擇器切換語言。
