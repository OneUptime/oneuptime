# 域名监控器

域名监控允许您监控域名的注册状态和过期情况。OneUptime 定期执行 WHOIS 查询，跟踪您的域名健康状况并在过期前提醒您。

## 概述

域名监控器查询 WHOIS 数据以跟踪您的域名注册详情。这使您能够：

- 监控域名过期日期
- 检测已过期或即将过期的域名
- 跟踪域名注册商信息
- 验证名称服务器配置
- 监控域名状态码

## 创建域名监控器

1. 在 OneUptime 控制台中转到 **监控器**
2. 点击 **创建监控器**
3. 选择 **域名** 作为监控器类型
4. 输入您要监控的域名
5. 根据需要配置监控标准

## 配置选项

### 基本设置

| 字段 | 描述                             | 是否必填 |
| ---- | -------------------------------- | -------- |
| 域名 | 要监控的域（例如 `example.com`） | 是       |

### 高级设置

| 字段         | 描述                  | 默认值 |
| ------------ | --------------------- | ------ |
| 超时（毫秒） | 等待 WHOIS 响应的时间 | 10000  |
| 重试次数     | 失败时的重试次数      | 3      |

## 监控标准

您可以配置标准来判断域名何时处于在线、降级或离线状态，基于以下条件：

### 可用检查类型

| 检查类型       | 描述                   |
| -------------- | ---------------------- |
| 域名过期天数   | 域名注册过期前的天数   |
| 域名注册商     | 域名注册商名称         |
| 域名服务器     | 域名的名称服务器主机名 |
| 域名状态码     | WHOIS 域名状态码       |
| 域名是否已过期 | 域名是否已过期         |

### 过滤类型

对于 **域名是否已过期**：

- **True** — 域名已过期
- **False** — 域名未过期

对于 **域名过期天数**：

- **大于**、**小于**、**大于或等于**、**小于或等于**、**等于**、**不等于**

对于 **域名注册商**、**域名服务器** 和 **域名状态码**：

- **包含** — 值包含指定文本
- **不包含** — 值不包含指定文本
- **以...开头** — 值以指定文本开头
- **以...结尾** — 值以指定文本结尾
- **等于** — 值完全匹配
- **不等于** — 值不匹配

### 示例标准

#### 当域名在 30 天内过期时发出告警

- **检查项**：域名过期天数
- **过滤类型**：小于
- **值**：30

#### 当域名已过期时标记为离线

- **检查项**：域名是否已过期
- **过滤类型**：True

#### 验证名称服务器是否正确

- **检查项**：域名服务器
- **过滤类型**：包含
- **值**：`ns1.example.com`

## 最佳实践

1. **设置提前预警** — 在过期前 60 天配置降级告警，在过期前 14 天配置离线告警
2. **监控所有关键域名** — 包括主要域名、单独注册的子域名，以及用于电子邮件或 API 的任何域名
3. **跟踪注册商变更** — 监控注册商字段以检测未经授权的域名转移
