# API 参考

OneUptime 提供全面的 REST API，允许您将监控、事件管理和状态页面功能集成到您的应用程序和工作流中。**您在 OneUptime 控制台上能做的任何事情，都可以通过 API 完成，从而实现自动化和自定义集成。**

### 入门

我们的 API 围绕 REST 原则构建，使用标准 HTTP 响应代码、认证和请求方法。所有 API 端点均返回 JSON 响应。

### 认证

所有 API 请求都需要使用 API 密钥进行认证。您可以在 OneUptime 控制台的设置 > API 密钥下生成 API 密钥。
