# Windows 安装指南

在 Windows 上将 OneUptime 安装为桌面应用程序，实现全面的监控和事件管理。

## 安装方法

### 方法一：Microsoft Edge（推荐）

Edge 提供最佳的 Windows PWA 集成，具有原生功能。

1. **在 Edge 中打开 OneUptime**

   - 启动 Microsoft Edge 浏览器
   - 导航至您的 OneUptime 实例 URL
   - 登录您的 OneUptime 账号
   - 等待页面完全加载

2. **安装应用**

   - 查找地址栏中的 **安装图标**（⊞）
   - 点击 **"安装 OneUptime"** 按钮
   - 或点击 **三点菜单** → **应用** → **将此站点安装为应用**

3. **自定义安装**

   - **应用名称**：根据需要修改（默认：OneUptime）
   - **开始菜单**：选择是否添加到开始菜单
   - **任务栏**：选择是否固定到任务栏
   - **桌面**：创建桌面快捷方式

4. **完成安装**
   - 点击 **"安装"** 完成
   - OneUptime 将在自己的窗口中打开
   - 在开始菜单的已安装应用下找到它

### 方法二：Google Chrome

Chrome 提供出色的 PWA 支持，具有丰富的桌面集成。

1. **在 Chrome 中打开 OneUptime**

   - 启动 Google Chrome
   - 前往您的 OneUptime 实例
   - 确保已登录
   - 允许页面完全加载

2. **通过地址栏安装**

   - 查找地址栏中的 **安装图标**（⊞）
   - 点击 **"安装 OneUptime"**
   - 或使用菜单：**三个点** → **更多工具** → **创建快捷方式**

3. **安装选项**

   - 勾选 **"作为窗口打开"** 以获得类应用体验
   - 根据需要自定义应用名称
   - 点击 **"安装"** 或 **"创建"**

4. **启动应用**
   - 在 Windows 开始菜单中找到 OneUptime
   - 或从桌面快捷方式启动
   - 应用在专用窗口中打开

### 方法三：Firefox

Firefox 支持 PWA 安装，具有基本的桌面集成。

1. **在 Firefox 中打开 OneUptime**

   - 启动 Firefox 浏览器
   - 导航至 OneUptime URL
   - 完成登录流程

2. **安装 PWA**
   - 查找 **安装提示** 或横幅
   - 或点击 **菜单** → **安装**
   - 如果可用，点击 **"添加到主屏幕"** 等效选项

### 启动配置

1. **自动启动**：配置 OneUptime 随 Windows 启动
   - 右键单击任务栏 → 任务管理器 → 启动
   - 根据需要启用 OneUptime
2. **默认大小**：设置首选窗口大小和位置

### 通知设置

1. **Windows 通知**

   - 设置 → 系统 → 通知和操作
   - 找到 OneUptime 并配置告警偏好
   - 为事件启用横幅通知

2. **专注助手**
   - 配置勿扰设置
   - 允许 OneUptime 紧急通知
   - 为不同告警类型设置优先级

## 故障排查

### 安装问题

**安装按钮不出现：**

```
解决方案：
1. 确保使用 Edge 或 Chrome（推荐浏览器）
2. 验证与 OneUptime 实例的 HTTPS 连接
3. 清除浏览器缓存和 Cookie
4. 将浏览器更新到最新版本
5. 检查服务器上是否满足 PWA 要求
6. 临时禁用浏览器扩展
```

**安装失败或崩溃：**

```
解决方案：
1. 以管理员身份运行浏览器
2. 检查 Windows 用户账户控制（UAC）设置
3. 确保足够的磁盘空间（最低 100MB）
4. 临时禁用防病毒软件
5. 完全清除浏览器数据
6. 重启 Windows 后重试
```

**应用未出现在开始菜单中：**

```
解决方案：
1. 在 Windows 搜索中搜索「OneUptime」
2. 检查是否以不同名称安装
3. 在"最近添加"应用部分查找
4. 重新安装并确保勾选"添加到开始菜单"
5. 如有必要，手动创建快捷方式
```

### 通知问题

**Windows 通知不工作：**

```
解决方案：
1. Windows 设置 → 系统 → 通知和操作
2. 为 OneUptime 启用通知
3. 检查专注助手设置
4. 确保 OneUptime 中的通知权限
5. 先测试简单通知
```

## 卸载

### 完全移除

1. **Windows 设置方法**

   - 设置 → 应用 → 应用和功能
   - 搜索「OneUptime」
   - 点击并选择"卸载"

2. **浏览器方法**

   - 打开 Edge/Chrome
   - 前往 edge://apps/ 或 chrome://apps/
   - 找到 OneUptime
   - 点击选项 → 卸载

3. **开始菜单方法**
   - 右键单击开始菜单中的 OneUptime
   - 选择"卸载"
   - 确认移除
