# 도메인 모니터

도메인 모니터링을 통해 도메인 이름의 등록 상태 및 만료를 모니터링할 수 있습니다. OneUptime은 주기적으로 WHOIS 조회를 수행하여 도메인 상태를 추적하고 만료 전에 알림을 보냅니다.

## 개요

도메인 모니터는 도메인의 WHOIS 데이터를 쿼리하여 등록 세부 정보를 추적합니다. 이를 통해 다음이 가능합니다:

- 도메인 만료 날짜 모니터링
- 만료되거나 곧 만료될 도메인 감지
- 도메인 등록 기관 정보 추적
- 네임서버 구성 확인
- 도메인 상태 코드 모니터링

## 도메인 모니터 생성

1. OneUptime 대시보드의 **모니터**로 이동합니다
2. **모니터 생성**을 클릭합니다
3. 모니터 유형으로 **도메인**을 선택합니다
4. 모니터링할 도메인 이름을 입력합니다
5. 필요에 따라 모니터링 기준을 구성합니다

## 구성 옵션

### 기본 설정

| 필드        | 설명                                  | 필수 여부 |
| ----------- | ------------------------------------- | --------- |
| 도메인 이름 | 모니터링할 도메인 (예: `example.com`) | 예        |

### 고급 설정

| 필드          | 설명                       | 기본값 |
| ------------- | -------------------------- | ------ |
| 타임아웃 (ms) | WHOIS 응답을 기다리는 시간 | 10000  |
| 재시도        | 실패 시 재시도 횟수        | 3      |

## 모니터링 기준

다음을 기반으로 도메인이 온라인, 저하 또는 오프라인으로 간주되는 시점을 결정하는 기준을 구성할 수 있습니다:

### 사용 가능한 확인 유형

| 확인 유형        | 설명                              |
| ---------------- | --------------------------------- |
| 도메인 만료 일수 | 도메인 등록이 만료되기까지의 일수 |
| 도메인 등록 기관 | 도메인 등록 기관 이름             |
| 도메인 네임서버  | 도메인의 네임서버 호스트 이름     |
| 도메인 상태 코드 | WHOIS 도메인 상태 코드            |
| 도메인 만료 여부 | 도메인이 만료되었는지 여부        |

### 필터 유형

**도메인 만료 여부**:

- **True** — 도메인이 만료됨
- **False** — 도메인이 만료되지 않음

**도메인 만료 일수**:

- **초과**, **미만**, **이상**, **이하**, **동일**, **다름**

**도메인 등록 기관**, **도메인 네임서버** 및 **도메인 상태 코드**:

- **포함** — 값에 지정된 텍스트가 포함됨
- **포함하지 않음** — 값에 지정된 텍스트가 포함되지 않음
- **시작** — 값이 지정된 텍스트로 시작함
- **끝** — 값이 지정된 텍스트로 끝남
- **동일** — 값이 정확히 일치함
- **다름** — 값이 일치하지 않음

### 기준 예시

#### 도메인이 30일 이내에 만료되는 경우 알림

- **확인 대상**: 도메인 만료 일수
- **필터 유형**: 미만
- **값**: 30

#### 도메인이 만료된 경우 오프라인으로 표시

- **확인 대상**: 도메인 만료 여부
- **필터 유형**: True

#### 네임서버가 올바른지 확인

- **확인 대상**: 도메인 네임서버
- **필터 유형**: 포함
- **값**: `ns1.example.com`

## 모범 사례

1. **조기 경고 설정** — 만료 60일 전에 저하 알림을, 14일 전에 오프라인 알림을 구성합니다
2. **모든 중요 도메인 모니터링** — 기본 도메인, 별도로 등록된 하위 도메인, 이메일이나 API에 사용되는 도메인을 포함합니다
3. **등록 기관 변경 추적** — 등록 기관 필드를 모니터링하여 무단 도메인 이전을 감지합니다
