# SSL 인증서 모니터

SSL 인증서 모니터링을 통해 웹사이트 및 서비스의 SSL/TLS 인증서의 유효성과 만료를 모니터링할 수 있습니다. OneUptime은 주기적으로 인증서를 확인하고 만료 전이나 문제가 감지될 때 알림을 보냅니다.

## 개요

SSL 인증서 모니터는 HTTPS 엔드포인트에 연결하고 SSL/TLS 인증서를 검사합니다. 이를 통해 다음이 가능합니다:

- 인증서 만료 날짜 모니터링
- 만료되거나 곧 만료될 인증서 감지
- 자체 서명된 인증서 식별
- 인증서 유효성 확인
- 만료된 인증서로 인한 서비스 장애 방지

## SSL 인증서 모니터 생성

1. OneUptime 대시보드의 **모니터**로 이동합니다
2. **모니터 생성**을 클릭합니다
3. 모니터 유형으로 **SSL 인증서**를 선택합니다
4. 확인할 HTTPS 엔드포인트의 URL을 입력합니다
5. 필요에 따라 모니터링 기준을 구성합니다

## 구성 옵션

### URL

SSL 인증서를 모니터링할 엔드포인트의 전체 HTTPS URL을 입력합니다 (예: `https://example.com` 또는 `https://example.com:8443`).

## 모니터링 기준

다음을 기반으로 인증서 상태가 온라인, 저하 또는 오프라인으로 간주되는 시점을 결정하는 기준을 구성할 수 있습니다:

### 사용 가능한 확인 유형

| 확인 유형                 | 설명                                                      |
| ------------------------- | --------------------------------------------------------- |
| 온라인 여부               | 서버에 도달 가능한지 여부                                 |
| 유효한 인증서 여부        | 인증서가 유효한지 여부 (만료되지 않고 자체 서명되지 않음) |
| 자체 서명된 인증서 여부   | 인증서가 자체 서명되었는지 여부                           |
| 만료된 인증서 여부        | 인증서가 만료되었는지 여부                                |
| 유효하지 않은 인증서 여부 | 인증서가 유효하지 않은지 여부                             |
| 만료까지 시간             | 인증서가 만료되기까지의 시간                              |
| 만료까지 일수             | 인증서가 만료되기까지의 일수                              |
| 요청 타임아웃 여부        | 연결이 타임아웃되었는지 여부                              |

### 필터 유형

**온라인 여부**, **유효한 인증서 여부**, **자체 서명된 인증서 여부**, **만료된 인증서 여부**, **유효하지 않은 인증서 여부** 및 **요청 타임아웃 여부**:

- **True** — 조건이 참임
- **False** — 조건이 거짓임

**만료까지 시간** 및 **만료까지 일수**:

- **초과** — 만료까지가 지정된 값보다 더 멀리 있음
- **미만** — 만료까지가 지정된 값보다 더 가까이 있음
- **이상** — 만료까지가 지정된 값 이상임
- **이하** — 만료까지가 지정된 값 이하임
- **동일** — 만료까지가 정확히 일치함
- **다름** — 만료까지가 일치하지 않음

### 기준 예시

#### 인증서가 30일 이내에 만료되는 경우 저하로 표시

- **확인 대상**: 만료까지 일수
- **필터 유형**: 미만
- **값**: 30

#### 인증서가 만료된 경우 오프라인으로 표시

- **확인 대상**: 만료된 인증서 여부
- **필터 유형**: True

#### 인증서가 자체 서명된 경우 알림

- **확인 대상**: 자체 서명된 인증서 여부
- **필터 유형**: True

#### 인증서가 유효하지 않은 경우 오프라인으로 표시

- **확인 대상**: 유효하지 않은 인증서 여부
- **필터 유형**: True

## 모범 사례

1. **여러 임계값 설정** — 만료 30일 전에는 저하 상태를, 7일 전에는 오프라인 상태를 사용하여 갱신할 시간을 충분히 확보합니다
2. **모든 엔드포인트 모니터링** — 여러 도메인이나 하위 도메인이 있는 경우 각각에 대한 모니터를 생성합니다
3. **비표준 포트 포함** — 비표준 포트에서 HTTPS를 실행하는 서비스를 잊지 마십시오
4. **갱신 후 모니터링** — 인증서를 갱신한 후 모니터가 유효하다고 확인하는지 검증합니다
