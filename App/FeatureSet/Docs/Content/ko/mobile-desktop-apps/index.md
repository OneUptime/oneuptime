# OneUptime 모바일 및 데스크탑 앱

OneUptime은 프로그레시브 웹 앱(PWA) 기술을 통해 네이티브와 같은 모바일 및 데스크탑 애플리케이션을 제공합니다. 이 앱들은 앱 스토어에서 다운로드할 필요 없이 모든 주요 플랫폼에서 원활한 경험을 제공합니다.

## 프로그레시브 웹 앱(PWA)이란?

프로그레시브 웹 앱은 사용자에게 앱과 같은 경험을 제공하기 위해 현대적인 웹 기능을 사용하는 웹 애플리케이션입니다.

## 시작하기

1. 지원되는 브라우저에서 OneUptime 인스턴스를 방문합니다
2. "설치" 또는 "홈 화면에 추가" 프롬프트를 찾습니다
3. 플랫폼별 설치 단계를 따릅니다
4. 기기의 앱 서랍이나 데스크탑에서 앱을 실행합니다

## 문제 해결

### 일반적인 문제

**설치 옵션이 나타나지 않는 경우:**
- 지원되는 브라우저를 사용하고 있는지 확인합니다
- OneUptime 인스턴스가 HTTPS를 통해 제공되는지 확인합니다
- 페이지를 새로 고침하거나 브라우저 캐시를 지우십시오

**푸시 알림이 작동하지 않는 경우:**
- 메시지가 표시되면 알림 권한을 부여합니다
- 브라우저 알림 설정을 확인합니다
- OneUptime 인스턴스에 알림이 구성되어 있는지 확인합니다


## 지원

OneUptime PWA에 문제가 발생한 경우:

1. 위의 문제 해결 섹션을 확인합니다
2. 브라우저가 최소 요구 사항을 충족하는지 확인합니다
3. OneUptime 대시보드를 통해 지원팀에 문의합니다
4. [GitHub 저장소](https://github.com/OneUptime/oneuptime)에서 문제를 보고합니다
