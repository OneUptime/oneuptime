# iOS 설치 가이드

이동 중에도 원활한 모니터링을 위해 iPhone 또는 iPad에 OneUptime을 네이티브 앱으로 설치합니다.

## 설치 방법

### 방법 1: Safari (권장)

Safari는 iOS 기기에서 최고의 PWA 경험을 제공합니다.

1. **Safari에서 OneUptime 열기**
   - iOS 기기에서 Safari를 실행합니다
   - OneUptime 인스턴스 URL로 이동합니다
   - 페이지가 완전히 로드될 때까지 기다립니다
   - OneUptime 계정에 로그인되어 있는지 확인합니다

2. **공유 메뉴 액세스**
   - 하단 도구 모음에서 **공유 버튼**(위쪽 화살표가 있는 사각형)을 탭합니다
   - 공유 옵션을 스크롤하여 "홈 화면에 추가"를 찾습니다

3. **홈 화면에 추가**
   - **"홈 화면에 추가"**를 탭합니다
   - 앱 이름을 커스터마이징합니다 (기본값: "OneUptime")
   - 오른쪽 상단의 **"추가"**를 탭합니다

4. **앱 실행**
   - 홈 화면에서 OneUptime 아이콘을 찾습니다
   - 탭하여 전체 화면 앱 모드로 실행합니다

### 방법 2: Chrome 브라우저

Chrome이 작동하지만 최상의 iOS PWA 경험을 위해 Safari가 권장됩니다.

1. **Chrome에서 OneUptime 열기**
   - Chrome 브라우저를 실행합니다
   - OneUptime 인스턴스로 이동합니다
   - 페이지가 완전히 로드될 때까지 기다립니다

2. **홈 화면에 추가**
   - **세 점 메뉴**(추가 옵션)를 탭합니다
   - **"홈 화면에 추가"**를 선택합니다
   - 원하는 경우 앱 이름을 커스터마이징합니다
   - **"추가"**를 탭합니다

### 방법 3: 기타 브라우저

Firefox, Edge 및 기타 브라우저는 기본 PWA 설치를 지원합니다:

1. **OneUptime 열기**
   - 원하는 브라우저를 실행합니다
   - OneUptime URL로 이동합니다
   - 페이지가 완전히 로드될 때까지 기다립니다

2. **설치 옵션 찾기**
   - "홈 화면에 추가" 또는 "설치"에 대한 브라우저 메뉴를 확인합니다
   - 브라우저별 설치 프롬프트를 따릅니다

### 커스터마이징 옵션

### 앱 아이콘 및 이름
- **커스텀 이름**: 설치 중 또는 이후에 변경합니다
- **아이콘 배치**: 폴더 또는 특정 홈 화면 페이지에 정리합니다
- **배지 알림**: 읽지 않은 인시던트 수를 표시합니다

### 알림 구성
1. **알림 활성화**
   - 메시지가 표시되면 알림에 대해 **"허용"**을 탭합니다
   - 또는 설정 → 알림 → OneUptime으로 이동합니다
   - 포괄적인 모니터링을 위해 모든 알림 유형을 활성화합니다

2. **알림 스타일 커스터마이징**
   - **잠금 화면**: 잠긴 기기에 인시던트 알림 표시
   - **배너 스타일**: 임시 또는 영구 배너 선택
   - **소리**: 알림 소리 및 진동 커스터마이징
   - **중요 알림**: 높은 우선 순위 인시던트에 활성화 (권한 필요)

## 문제 해결

### 설치 문제

**"홈 화면에 추가"가 표시되지 않는 경우:**
```
해결책:
1. Safari를 사용하고 있는지 확인합니다 (최상의 호환성)
2. 페이지를 새로 고침하고 30초 기다립니다
3. 올바른 OneUptime URL에 있는지 확인합니다
4. HTTPS 연결을 확인합니다 (자물쇠 아이콘 확인)
5. Safari 캐시 지우기: 설정 → Safari → 방문 기록 및 웹 사이트 데이터 지우기
```

**설치가 완료되었지만 아이콘이 나타나지 않는 경우:**
```
해결책:
1. 모든 홈 화면 페이지를 확인합니다
2. 앱 라이브러리를 확인합니다 (마지막 홈 화면 페이지를 지나 왼쪽으로 스와이프)
3. "OneUptime"을 찾기 위해 Spotlight 검색을 사용합니다
4. 기기를 다시 시작하고 다시 확인합니다
5. 필요한 경우 재설치합니다
```

**실행 시 앱이 충돌하는 경우:**
```
해결책:
1. 앱을 강제로 닫고 다시 엽니다
2. iOS 기기를 다시 시작합니다
3. Safari 캐시를 지우고 재설치합니다
4. iOS 버전이 11.3 이상인지 확인합니다
5. 기기 저장 공간을 확보합니다
```

### 알림 문제

**푸시 알림을 받지 못하는 경우:**
```
다음 설정을 확인합니다:
1. 설정 → 알림 → OneUptime → 알림 허용
2. 설정 → Screen Time → 콘텐츠 및 개인 정보 보호 제한 → 허용된 앱
3. 방해 금지 설정
4. OneUptime 대시보드에서 알림 설정 확인
5. OneUptime에서 로그아웃 후 다시 로그인
```

**지연되거나 누락된 알림:**
```
해결책:
1. 앱을 백그라운드에서 실행 상태로 유지합니다 (강제로 닫지 않음)
2. 중요한 모니터링 중에는 저전력 모드를 비활성화합니다
3. 백그라운드 앱 새로 고침 확인: 설정 → 일반 → 백그라운드 앱 새로 고침
4. 충분한 저장 공간이 있는지 확인합니다
```

## 제거

### 홈 화면에서 제거
1. OneUptime 앱 아이콘을 **길게 누릅니다**
2. **"앱 제거"**를 탭합니다
3. **"앱 삭제"**를 선택합니다
4. 삭제를 확인합니다

### 대체 방법
1. **설정 → 일반 → iPhone 저장 공간**으로 이동합니다
2. 앱 목록에서 **OneUptime**을 찾습니다
3. **"앱 삭제"**를 탭합니다
4. 제거를 확인합니다

## 업데이트 및 유지 관리

### 자동 업데이트
- OneUptime PWA는 온라인 상태에서 자동으로 업데이트됩니다
- App Store 업데이트가 필요하지 않습니다
- 서버 배포 후 즉시 새 기능을 사용할 수 있습니다
- 중요한 보안 업데이트가 즉시 적용됩니다

## iPad 특정 설치

### 향상된 iPad 경험
1. **더 큰 인터페이스**: iPad 화면 크기에 최적화된 레이아웃
2. **멀티 윈도우**: 여러 OneUptime 창을 동시에 실행
3. **키보드 단축키**: 외부 키보드에 대한 완전한 지원
4. **드래그 앤 드롭**: OneUptime과 다른 앱 사이에서 데이터 이동

### iPad 설치 단계
iPhone 설치와 동일하지만 추가적인 고려 사항이 있습니다:
- 최적의 대시보드 보기를 위해 가로 모드 사용
- 다른 생산성 앱과 함께 Split View 설정 고려
- 일반적인 작업에 대한 키보드 단축키 구성

## Apple Watch 통합

OneUptime에는 전용 watchOS 앱이 없지만 다음이 가능합니다:
- **알림 받기**: Apple Watch에 인시던트 알림이 표시됩니다
- **빠른 작업**: Watch 알림에서 인시던트 확인
- **Siri 통합**: 시스템 상태에 대해 Siri에게 질문 (구성된 경우)

## 고급 구성

### 단축어 앱 통합
OneUptime에 대한 커스텀 Siri 단축어 생성:
1. **단축어** 앱 열기
2. **새 단축어** 생성
3. **"앱 열기"** 작업 추가
4. **OneUptime** 선택
5. "시스템 상태 확인"과 같은 음성 구문 추가

### 집중 모드
iOS 집중 모드와 OneUptime 통합:
1. **설정 → 집중**
2. 집중 모드 선택 또는 생성
3. **앱 → 앱 추가 → OneUptime**
4. 다양한 집중 상태에 대한 알림 동작 구성

### Screen Time 관리
적절한 Screen Time 설정 구성:
1. **설정 → Screen Time → 앱 제한**
2. OneUptime을 "생산성" 카테고리에 추가
3. 필요한 경우 적절한 사용 제한 설정

## 모범 사례

### 보안 권장 사항
1. **URL 확인**: 조직의 공식 OneUptime 인스턴스에서만 설치합니다
2. **HTTPS만 사용**: 보안 연결 확인 (자물쇠 아이콘 확인)
3. **정기적인 업데이트**: 보안 패치를 위해 iOS를 최신 상태로 유지합니다
4. **앱 권한**: 필요한 권한만 부여합니다

### 성능 최적화
1. **Wi-Fi 설치**: 초기 설치 및 주요 업데이트에 Wi-Fi 사용
2. **백그라운드 새로 고침**: 적시 알림을 위해 활성화
3. **저장 관리**: 충분한 여유 공간 유지
4. **정기적인 재시작**: 최적의 성능을 위해 주 1회 앱 재시작

### 모니터링 모범 사례
1. **중요 알림**: 높은 우선 순위 알림에만 활성화
2. **여러 기기**: 중복성을 위해 iPhone과 iPad 모두에 설치
3. **팀 액세스**: 팀 구성원과 설치 가이드 공유
4. **테스트**: 정기적으로 알림 전달 및 오프라인 기능 테스트
