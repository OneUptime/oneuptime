# Android 설치 가이드

최상의 모니터링 경험을 위해 Android 기기에 OneUptime을 네이티브 앱으로 설치합니다.

## 설치 방법

### 방법 1: Chrome 브라우저 (권장)

1. **Chrome에서 OneUptime 열기**
   - Android 기기에서 Google Chrome을 실행합니다
   - OneUptime 인스턴스 URL로 이동합니다
   - 페이지가 완전히 로드될 때까지 기다립니다

2. **설치 프롬프트**
   - 하단의 "홈 화면에 추가" 배너를 찾습니다
   - "설치" 또는 "홈 화면에 추가"를 탭합니다
   - 프롬프트가 표시되지 않으면 오른쪽 상단의 세 점 메뉴(⋮)를 탭합니다

3. **메뉴를 통한 수동 설치**
   - Chrome 메뉴(세 점)를 탭합니다
   - "홈 화면에 추가" 또는 "앱 설치"를 선택합니다
   - 원하는 경우 앱 이름을 커스터마이징합니다
   - "추가"를 탭하여 확인합니다

4. **앱 실행**
   - 홈 화면이나 앱 서랍에서 OneUptime 아이콘을 찾습니다
   - 탭하여 전체 화면 모드로 앱을 실행합니다

### 방법 2: Samsung Internet

1. **OneUptime 열기**
   - Samsung Internet 브라우저를 실행합니다
   - OneUptime 인스턴스로 이동합니다
   - 페이지가 완전히 로드될 때까지 기다립니다

2. **홈 화면에 추가**
   - 메뉴 버튼(세 줄)을 탭합니다
   - "페이지 추가" → "홈 화면"을 선택합니다
   - 앱 이름을 입력하고 "추가"를 탭합니다

3. **실행**
   - 홈 화면에서 앱 아이콘을 찾습니다
   - 탭하여 앱 모드로 OneUptime을 엽니다

### 방법 3: Firefox

1. **OneUptime 열기**
   - Firefox 브라우저를 실행합니다
   - OneUptime URL로 이동합니다
   - 페이지가 완전히 로드될 때까지 기다립니다

2. **설치**
   - 세 점 메뉴를 탭합니다
   - "설치"를 선택합니다 (사용 가능한 경우)
   - 또는 "홈 화면에 추가"를 선택합니다
   - 설치를 확인합니다

### 커스터마이징 옵션

### 앱 이름
- 설치 중에 앱 이름을 커스터마이징할 수 있습니다
- 기본값: "OneUptime"
- 권장: "OneUptime"으로 유지하거나 회사 이름을 추가합니다

### 알림 설정
1. **권한 부여**
   - 메시지가 표시되면 알림을 허용합니다
   - 설정 → 앱 → OneUptime → 알림으로 이동합니다
   - 최상의 경험을 위해 모든 알림 카테고리를 활성화합니다

2. **알림 커스터마이징**
   - 어떤 인시던트가 알림을 트리거하는지 구성합니다
   - 알림 우선 순위 수준을 설정합니다
   - 소리 및 진동 설정을 선택합니다

## 문제 해결

### 설치 문제

**"홈 화면에 추가"가 나타나지 않는 경우:**
```
1. 브라우저 캐시 및 쿠키를 지웁니다
2. HTTPS (보안 연결)를 사용하고 있는지 확인합니다
3. 프롬프트를 찾기 전에 페이지에서 2-3분 기다립니다
4. OneUptime 인스턴스의 PWA 요구 사항이 충족되었는지 확인합니다
```

**설치 실패:**
```
1. 저장 공간을 확보합니다 (최소 50MB 필요)
2. 브라우저를 최신 버전으로 업데이트합니다
3. 브라우저를 다시 시작하고 다시 시도합니다
4. 다른 브라우저를 시도합니다 (Chrome 권장)
```

**앱 아이콘이 나타나지 않는 경우:**
```
1. 홈 화면과 앱 서랍을 확인합니다
2. "최근에 추가된" 앱 섹션을 확인합니다
3. 앱 서랍에서 "OneUptime"을 검색합니다
4. 필요한 경우 재설치합니다
```

### 알림 문제

**알림을 받지 못하는 경우:**
```
1. 알림 권한을 확인합니다:
   - 설정 → 앱 → OneUptime → 권한 → 알림
2. OneUptime 대시보드에서 알림이 활성화되어 있는지 확인합니다
3. 방해 금지 설정을 확인합니다
4. 배터리 최적화 설정이 OneUptime을 차단하지 않는지 확인합니다
```

**알림이 지연되는 경우:**
```
1. OneUptime에 대한 배터리 최적화를 비활성화합니다:
   - 설정 → 앱 → OneUptime → 배터리 → 배터리 사용량 최적화
2. 백그라운드 활동을 허용합니다
3. 데이터 절약 설정을 확인합니다
```

## 제거

### 앱 제거
1. 홈 화면에서 OneUptime 아이콘을 **길게 누릅니다**
2. **"제거"**를 선택하거나 휴지통으로 드래그합니다
3. 제거를 확인합니다

### 대체 방법
1. **설정 → 앱**으로 이동합니다
2. **"OneUptime"**을 찾습니다
3. **"제거"**를 탭합니다
4. 제거를 확인합니다

## 업데이트 및 유지 관리

### 자동 업데이트
OneUptime PWA는 자동으로 업데이트됩니다:
- **자동 업데이트**: 온라인 상태에서 방문할 때 앱이 업데이트됩니다
- **수동 업데이트 없음**: 스토어 앱과 달리 사용자 작업이 필요하지 않습니다
- **즉시 업데이트**: 새 기능을 즉시 사용할 수 있습니다
- **롤백 안전**: 잘못된 업데이트는 빠르게 되돌릴 수 있습니다

## 고급 구성

### 개발자 옵션
PWA를 검사하려는 고급 사용자의 경우:
1. Android에서 개발자 옵션을 활성화합니다
2. ADB를 사용하여 컴퓨터에 연결합니다
3. 원격 디버깅에 Chrome DevTools를 사용합니다

### 네트워크 구성
- 내부 OneUptime 인스턴스에 액세스하는 경우 VPN을 구성합니다
- 조직에서 요구하는 경우 프록시 설정을 설정합니다
- 방화벽에서 PWA 리소스를 허용하는지 확인합니다

## 모범 사례

### 최적의 성능을 위해
1. **첫 번째 실행**: 초기 설정을 위해 항상 온라인 상태여야 합니다
2. **일반 사용**: 캐시를 최신 상태로 유지하려면 앱을 정기적으로 엽니다
3. **저장 관리**: 충분한 여유 공간을 유지합니다
4. **네트워크**: 초기 설치 및 주요 업데이트에는 Wi-Fi를 사용합니다

### 보안 권장 사항
1. **HTTPS만 사용**: 보안 OneUptime 인스턴스에서만 설치합니다
2. **공식 URL**: 조직의 공식 OneUptime URL에서 설치하는지 확인합니다
3. **권한**: 필요한 권한만 부여합니다
4. **업데이트**: Android OS 및 브라우저를 최신 상태로 유지합니다
