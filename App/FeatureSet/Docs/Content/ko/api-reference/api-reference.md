# API 참조

OneUptime은 모니터링, 인시던트 관리 및 상태 페이지 기능을 애플리케이션과 워크플로에 통합할 수 있는 포괄적인 REST API를 제공합니다. **OneUptime 대시보드에서 할 수 있는 모든 작업은 API를 통해서도 수행할 수 있어 자동화와 커스텀 통합이 가능합니다.**

### 시작하기

당사의 API는 REST 원칙을 중심으로 구성되어 있으며 표준 HTTP 응답 코드, 인증 및 동사를 사용합니다. 모든 API 엔드포인트는 JSON 응답을 반환합니다.


### 인증

모든 API 요청은 API 키를 사용한 인증이 필요합니다. OneUptime 대시보드의 설정 > API 키에서 API 키를 생성할 수 있습니다.
