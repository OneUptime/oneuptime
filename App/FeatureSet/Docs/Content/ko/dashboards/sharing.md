# 공유 및 공개 대시보드

기본적으로 대시보드는 프로젝트에서 비공개로, 로그인한 팀원만 볼 수 있습니다. 하지만 OneUptime은 대시보드를 공개적으로 공유하고, 비밀번호로 보호하며, 특정 IP로 제한하고, 자체 도메인에서 호스팅하는 것도 허용합니다. 이 페이지는 이 네 가지를 모두 다룹니다.

## 비공개 대시보드(기본값)

대시보드는 로그인한 프로젝트 멤버만 접근할 수 있습니다. URL은 `https://oneuptime.com/dashboards/<id>/view` 형태이며 로그인이 필요합니다.

프로젝트 내에서 소유자와 라벨이 누가 무엇을 보는지 제어합니다. [구성 및 권한](/docs/dashboards/configuration)을 참조하시기 바랍니다.

## 공개 대시보드

**Dashboard → Settings**에서 **Public Dashboard**를 켭니다. 이제 대시보드는 로그인이 필요 없는 두 번째 URL을 가집니다. 벤더, 파트너, 고객과 공유하거나 공개 README에 붙여넣을 수 있습니다.

공개 대시보드는 다음과 같습니다.

- 항상 **View** 모드에서 열립니다. 공개 방문자는 편집하거나 위젯 팔레트를 볼 수 없습니다.
- 추가한 변수가 포함됩니다. 방문자는 팀이 사용하는 것과 동일한 드롭다운에서 선택합니다.
- Settings에서 설정한 **브랜딩**(페이지 제목, 설명, 로고, 파비콘)을 사용합니다.

공개 대시보드 활성화는 웹페이지 게시와 같이 취급하시기 바랍니다. 모든 위젯이 전 세계에 공개됩니다. 스위치를 켜기 전에 캔버스에 무엇이 있는지 확인하시기 바랍니다.

## 마스터 비밀번호

공개 대시보드에 비밀번호를 설정하려면 다음과 같이 합니다.

1. **Public Dashboard**를 켭니다.
2. **Master Password**를 켭니다.
3. 비밀번호를 설정합니다.

방문자는 대시보드가 나타나기 전에 비밀번호 프롬프트를 보게 됩니다. 비밀번호는 해시로 저장되며, 실제 비밀번호는 절대 볼 수 없습니다.

다음과 같은 경우 마스터 비밀번호를 사용하시기 바랍니다.

- 파트너나 고객과 공유하고 싶지만 URL이 유출되어도 유용하지 않게 하고 싶을 때.
- 대시보드가 "반공개"인 경우 — 모든 보는 사람을 팀 멤버로 초대하고 싶지는 않지만 공개 인터넷에 두기에는 너무 개방적인 경우.

더 강력한 게이팅(보는 사람별 별도 계정, 누가 무엇을 봤는지에 대한 감사 추적)이 필요하다면 대시보드를 비공개로 유지하고 보는 사람을 읽기 전용 팀원으로 초대하시기 바랍니다.

## IP 허용 목록

**Scale** 요금제에서는 공개 대시보드를 IP 주소 또는 범위 목록으로 제한할 수 있습니다. **Dashboard → Settings → IP Whitelist**에서 구성합니다.

다음과 같은 경우에 사용하시기 바랍니다.

- 대시보드가 사무실이나 VPN에서만 접근 가능해야 할 때.
- 벤더 포털이 알려진 IP에서만 접근 가능해야 할 때.
- 마스터 비밀번호 위에 추가 보호를 원할 때.

다른 IP의 요청은 거부됩니다.

## 커스텀 도메인

기본적으로 공개 대시보드는 `oneuptime.com`에서 제공됩니다. `dashboard.acme.com`과 같은 자체 서브도메인에서 호스팅하려면 다음과 같이 합니다.

1. DNS에 CNAME 레코드를 추가하여 서브도메인을 OneUptime의 타겟으로 가리키게 합니다.
2. **Dashboard → Settings → Custom Domains**에서 도메인을 추가합니다.
3. 확인합니다. OneUptime이 DNS 레코드를 확인해 드립니다.
4. 확인 후, 대시보드는 커스텀 도메인과 원래 URL 모두에서 접근 가능해집니다.

커스텀 도메인은 다음과 같은 경우에 유용합니다.

- 자체 브랜드의 고객 대상 대시보드.
- 공동 브랜드 파트너 대시보드.
- 자체 URL을 가진 공개 상태 페이지.

여러 대상에 동일한 콘텐츠를 제공한다면 하나의 대시보드에 둘 이상의 커스텀 도메인을 연결할 수 있습니다.

## 브랜딩

**Dashboard → Settings**에서 다음을 구성할 수 있습니다.

- **Page title** — 브라우저 탭과 페이지 상단에 표시됩니다.
- **Page description** — 검색 엔진과 소셜 미리보기에 사용되는 설명.
- **Logo** — 헤더에 표시할 PNG 또는 SVG를 업로드합니다.
- **Favicon** — 브라우저 탭의 작은 아이콘.

브랜딩은 대시보드를 공개적으로 볼 때만 적용됩니다. 내부 사용자는 항상 OneUptime의 브랜딩을 보게 됩니다.

## 임베딩

iframe을 사용해 자체 사이트에 공개 대시보드를 임베드할 수 있습니다.

```html
<iframe src="https://dashboard.acme.com/view"
        width="100%" height="800"
        frameborder="0"></iframe>
```

대시보드에 마스터 비밀번호가 있다면 방문자는 iframe 내에서 비밀번호 프롬프트를 보게 됩니다.

## 공유 가능한 URL

대시보드 URL에는 현재 변수 선택과 시간 범위가 쿼리 파라미터로 포함됩니다. 드롭다운을 조정하고 URL을 복사하여 채팅에 붙여넣으세요. 링크를 여는 사람은 정확히 동일한 뷰의 대시보드를 보게 됩니다.

이것은 "인시던트가 시작된 시점의 대시보드"로 팀원을 안내하는 가장 빠른 방법입니다. 시간 범위를 고정하고, 복사하고, 붙여넣으시면 됩니다.
