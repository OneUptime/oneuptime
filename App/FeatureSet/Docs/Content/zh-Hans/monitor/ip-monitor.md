# IP 监控器

IP 监控允许您监控任何 IPv4 或 IPv6 地址的可用性和响应能力。OneUptime 定期测试与目标 IP 地址的连接性并报告其状态。

## 概述

IP 监控器验证特定 IP 地址是否可达且响应正常。这使您能够：

- 监控 IPv4 和 IPv6 地址可用性
- 跟踪响应时间和延迟
- 检测网络连接问题
- 验证基础设施端点是否可达

## 创建 IP 监控器

1. 在 OneUptime 控制台中转到 **监控器**
2. 点击 **创建监控器**
3. 选择 **IP** 作为监控器类型
4. 输入您要监控的 IP 地址
5. 根据需要配置监控标准

## 配置选项

### IP 地址

输入您要监控的 IPv4 或 IPv6 地址（例如 `192.168.1.1` 或 `2001:db8::1`）。该值必须是有效的 IP 地址格式。

## 监控标准

您可以配置标准来判断 IP 地址何时处于在线、降级或离线状态，基于以下条件：

### 可用检查类型

| 检查类型 | 描述 |
|---------|------|
| 是否在线 | IP 地址是否可达 |
| 响应时间（毫秒） | 响应时间（毫秒） |
| 请求是否超时 | 请求是否超时 |

### 过滤类型

对于 **是否在线** 和 **请求是否超时**：

- **True** — 条件为真
- **False** — 条件为假

对于 **响应时间**：

- **大于** — 响应时间超过阈值
- **小于** — 响应时间低于阈值
- **大于或等于** — 响应时间等于或超过阈值
- **小于或等于** — 响应时间等于或低于阈值
- **等于** — 响应时间完全匹配
- **不等于** — 响应时间不匹配
- **随时间评估** — 使用聚合（平均值、求和、最大值、最小值、所有值、任意值）在时间窗口内评估

### 示例标准

#### 如果 IP 不可达则标记为离线

- **检查项**：是否在线
- **过滤类型**：False

#### 如果延迟超过 100ms 则发出告警

- **检查项**：响应时间（毫秒）
- **过滤类型**：大于
- **值**：100
