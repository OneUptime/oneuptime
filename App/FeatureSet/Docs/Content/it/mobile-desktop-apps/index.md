# App Mobile e Desktop di OneUptime

OneUptime offre applicazioni mobile e desktop simili a quelle native attraverso la tecnologia Progressive Web App (PWA). Queste app forniscono un'esperienza fluida su tutte le principali piattaforme senza richiedere il download dagli app store.

## Cos'è una Progressive Web App (PWA)?

Una Progressive Web App è un'applicazione web che utilizza le moderne funzionalità web per offrire agli utenti un'esperienza simile a quella di un'app.

## Iniziare

1. Visita la tua istanza OneUptime in un browser supportato
2. Cerca il prompt "Installa" o "Aggiungi alla schermata Home"
3. Segui i passaggi di installazione specifici per la piattaforma
4. Avvia l'app dal cassetto delle app del tuo dispositivo o dal desktop

## Risoluzione dei Problemi

### Problemi Comuni

**L'opzione di installazione non appare:**
- Assicurati di usare un browser supportato
- Controlla che la tua istanza OneUptime sia servita tramite HTTPS
- Prova ad aggiornare la pagina o a svuotare la cache del browser

**Le notifiche push non funzionano:**
- Concedi i permessi di notifica quando richiesto
- Controlla le impostazioni delle notifiche del browser
- Assicurati che l'istanza OneUptime abbia le notifiche configurate
