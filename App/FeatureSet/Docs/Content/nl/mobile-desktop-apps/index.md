# OneUptime Mobiele en Desktopapps

OneUptime biedt native-achtige mobiele en desktopapplicaties via Progressive Web App (PWA)-technologie. Deze apps bieden een naadloze ervaring op alle grote platformen zonder dat downloads uit app stores vereist zijn.

## Wat is een Progressive Web App (PWA)?

Een Progressive Web App is een webapplicatie die gebruikmaakt van moderne webmogelijkheden om een app-achtige ervaring aan gebruikers te bieden.

## Aan de slag

1. Bezoek uw OneUptime-instantie in een ondersteunde browser
2. Zoek naar de melding "Installeren" of "Aan startscherm toevoegen"
3. Volg de platformspecifieke installatiestappen
4. Start de app vanuit het app-startprogramma of de desktop van uw apparaat

## Probleemoplossing

### Veelvoorkomende problemen

**Installatieoptie verschijnt niet:**
- Zorg dat u een ondersteunde browser gebruikt
- Controleer of uw OneUptime-instantie via HTTPS wordt aangeboden
- Probeer de pagina te vernieuwen of de browsercache te wissen

**Push-meldingen werken niet:**
- Verleen meldingsmachtigingen wanneer daarom wordt gevraagd
- Controleer de meldingsinstellingen van de browser
- Zorg dat de OneUptime-instantie meldingen heeft geconfigureerd


## Ondersteuning

Als u problemen ondervindt met de OneUptime PWA:

1. Controleer de bovenstaande sectie voor probleemoplossing
2. Verifieer dat uw browser voldoet aan de minimumvereisten
3. Neem contact op met ondersteuning via uw OneUptime-dashboard
4. Meld problemen op onze [GitHub-repository](https://github.com/OneUptime/oneuptime)
