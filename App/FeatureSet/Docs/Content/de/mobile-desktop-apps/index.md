# OneUptime Mobile- und Desktop-Apps

OneUptime bietet native-ähnliche Mobile- und Desktop-Anwendungen durch Progressive Web App (PWA)-Technologie. Diese Apps bieten ein nahtloses Erlebnis auf allen wichtigen Plattformen, ohne Downloads aus App-Stores zu erfordern.

## Was ist eine Progressive Web App (PWA)?

Eine Progressive Web App ist eine Webanwendung, die moderne Web-Fähigkeiten nutzt, um Benutzern ein app-ähnliches Erlebnis zu bieten.

## Erste Schritte

1. Besuchen Sie Ihre OneUptime-Instanz in einem unterstützten Browser
2. Suchen Sie nach der Aufforderung „Installieren" oder „Zum Startbildschirm hinzufügen"
3. Befolgen Sie die plattformspezifischen Installationsschritte
4. Starten Sie die App aus der App-Übersicht oder dem Desktop Ihres Geräts

## Fehlerbehebung

### Häufige Probleme

**Installationsoption erscheint nicht:**
- Stellen Sie sicher, dass Sie einen unterstützten Browser verwenden
- Prüfen Sie, ob Ihre OneUptime-Instanz über HTTPS bereitgestellt wird
- Versuchen Sie, die Seite zu aktualisieren oder den Browser-Cache zu leeren

**Push-Benachrichtigungen funktionieren nicht:**
- Erteilen Sie Benachrichtigungsberechtigungen, wenn Sie dazu aufgefordert werden
- Prüfen Sie die Browser-Benachrichtigungseinstellungen
- Stellen Sie sicher, dass die OneUptime-Instanz Benachrichtigungen konfiguriert hat


## Support

Wenn Sie Probleme mit der OneUptime PWA haben:

1. Prüfen Sie den Abschnitt zur Fehlerbehebung oben
2. Stellen Sie sicher, dass Ihr Browser die Mindestanforderungen erfüllt
3. Kontaktieren Sie den Support über Ihr OneUptime-Dashboard
4. Melden Sie Probleme in unserem [GitHub-Repository](https://github.com/OneUptime/oneuptime)
