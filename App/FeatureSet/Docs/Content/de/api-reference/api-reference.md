# API-Referenz

OneUptime bietet eine umfassende REST API, mit der Sie Überwachungs-, Incident-Management- und Status-Seiten-Funktionalität in Ihre Anwendungen und Workflows integrieren können. **Alles, was Sie im OneUptime-Dashboard tun können, lässt sich auch über die API erledigen und ermöglicht so Automatisierung und benutzerdefinierte Integrationen.**

### Erste Schritte

Unsere API ist nach REST-Prinzipien aufgebaut und verwendet standardmäßige HTTP-Antwortcodes, Authentifizierung und Verben. Alle API-Endpunkte geben JSON-Antworten zurück.

### Authentifizierung

Alle API-Anfragen erfordern eine Authentifizierung über API-Schlüssel. Sie können API-Schlüssel in Ihrem OneUptime-Dashboard unter Einstellungen > API-Schlüssel generieren.
