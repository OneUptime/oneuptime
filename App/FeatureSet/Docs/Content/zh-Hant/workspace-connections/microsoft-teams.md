# 將 OneUptime 連接到 Microsoft Teams

### 連接 OneUptime 到 Microsoft Teams 的步驟

1. **在 OneUptime 上創建賬號**
    - 訪問 [OneUptime.com](https://oneuptime.com) 並創建賬號。
    - 創建賬號後，創建一個新項目。

2. **將 Microsoft Teams 連接到 OneUptime 項目**
    - 在您的 OneUptime 項目中，導航至 **項目設置** > **Microsoft Teams**。
    - 按照提示將您的 Microsoft Teams 賬號與 OneUptime 項目連接。

3. **配置事件通知**
    - 連接 Microsoft Teams 賬號後，前往 **事件頁面** > **Microsoft Teams**。
    - 添加規則以將事件通知發送到 Microsoft Teams。例如，您可以創建一條規則，當創建事件時向 Teams 頻道發佈消息。

4. **配置警報和計劃維護通知**
    - 類似的規則也可以應用於警報和計劃維護，方法是導航到各自的頁面並配置所需規則。
