# IP 監控器

IP 監控允許您監控任何 IPv4 或 IPv6 地址的可用性和響應能力。OneUptime 定期測試與目標 IP 地址的連接性並報告其狀態。

## 概述

IP 監控器驗證特定 IP 地址是否可達且響應正常。這使您能夠：

- 監控 IPv4 和 IPv6 地址可用性
- 跟蹤響應時間和延遲
- 檢測網絡連接問題
- 驗證基礎設施端點是否可達

## 創建 IP 監控器

1. 在 OneUptime 控制台中轉到 **監控器**
2. 點擊 **創建監控器**
3. 選擇 **IP** 作爲監控器類型
4. 輸入您要監控的 IP 地址
5. 根據需要配置監控標準

## 配置選項

### IP 地址

輸入您要監控的 IPv4 或 IPv6 地址（例如 `192.168.1.1` 或 `2001:db8::1`）。該值必須是有效的 IP 地址格式。

## 監控標準

您可以配置標準來判斷 IP 地址何時處於在線、降級或離線狀態，基於以下條件：

### 可用檢查類型

| 檢查類型 | 描述 |
|---------|------|
| 是否在線 | IP 地址是否可達 |
| 響應時間（毫秒） | 響應時間（毫秒） |
| 請求是否超時 | 請求是否超時 |

### 過濾類型

對於 **是否在線** 和 **請求是否超時**：

- **True** — 條件爲真
- **False** — 條件爲假

對於 **響應時間**：

- **大於** — 響應時間超過閾值
- **小於** — 響應時間低於閾值
- **大於或等於** — 響應時間等於或超過閾值
- **小於或等於** — 響應時間等於或低於閾值
- **等於** — 響應時間完全匹配
- **不等於** — 響應時間不匹配
- **隨時間評估** — 使用聚合（平均值、求和、最大值、最小值、所有值、任意值）在時間窗口內評估

### 示例標準

#### 如果 IP 不可達則標記爲離線

- **檢查項**：是否在線
- **過濾類型**：False

#### 如果延遲超過 100ms 則發出警報

- **檢查項**：響應時間（毫秒）
- **過濾類型**：大於
- **值**：100
