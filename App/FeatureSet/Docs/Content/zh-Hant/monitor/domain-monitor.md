# 域名監控器

域名監控允許您監控域名的註冊狀態和過期情況。OneUptime 定期執行 WHOIS 查詢，跟蹤您的域名健康狀況並在過期前提醒您。

## 概述

域名監控器查詢 WHOIS 數據以跟蹤您的域名註冊詳情。這使您能夠：

- 監控域名過期日期
- 檢測已過期或即將過期的域名
- 跟蹤域名註冊商信息
- 驗證名稱服務器配置
- 監控域名狀態碼

## 創建域名監控器

1. 在 OneUptime 控制台中轉到 **監控器**
2. 點擊 **創建監控器**
3. 選擇 **域名** 作爲監控器類型
4. 輸入您要監控的域名
5. 根據需要配置監控標準

## 配置選項

### 基本設置

| 字段 | 描述 | 是否必填 |
|------|------|---------|
| 域名 | 要監控的域（例如 `example.com`） | 是 |

### 高級設置

| 字段 | 描述 | 默認值 |
|------|------|--------|
| 超時（毫秒） | 等待 WHOIS 響應的時間 | 10000 |
| 重試次數 | 失敗時的重試次數 | 3 |

## 監控標準

您可以配置標準來判斷域名何時處於在線、降級或離線狀態，基於以下條件：

### 可用檢查類型

| 檢查類型 | 描述 |
|---------|------|
| 域名過期天數 | 域名註冊過期前的天數 |
| 域名註冊商 | 域名註冊商名稱 |
| 域名服務器 | 域名的名稱服務器主機名 |
| 域名狀態碼 | WHOIS 域名狀態碼 |
| 域名是否已過期 | 域名是否已過期 |

### 過濾類型

對於 **域名是否已過期**：

- **True** — 域名已過期
- **False** — 域名未過期

對於 **域名過期天數**：

- **大於**、**小於**、**大於或等於**、**小於或等於**、**等於**、**不等於**

對於 **域名註冊商**、**域名服務器** 和 **域名狀態碼**：

- **包含** — 值包含指定文本
- **不包含** — 值不包含指定文本
- **以...開頭** — 值以指定文本開頭
- **以...結尾** — 值以指定文本結尾
- **等於** — 值完全匹配
- **不等於** — 值不匹配

### 示例標準

#### 當域名在 30 天內過期時發出警報

- **檢查項**：域名過期天數
- **過濾類型**：小於
- **值**：30

#### 當域名已過期時標記爲離線

- **檢查項**：域名是否已過期
- **過濾類型**：True

#### 驗證名稱服務器是否正確

- **檢查項**：域名服務器
- **過濾類型**：包含
- **值**：`ns1.example.com`

## 最佳實踐

1. **設置提前預警** — 在過期前 60 天配置降級警報，在過期前 14 天配置離線警報
2. **監控所有關鍵域名** — 包括主要域名、單獨註冊的子域名，以及用於電郵或 API 的任何域名
3. **跟蹤註冊商變更** — 監控註冊商字段以檢測未經授權的域名轉移
