# Windows 安裝指南

在 Windows 上將 OneUptime 安裝爲桌面應用程序，實現全面的監控和事件管理。


## 安裝方法

### 方法一：Microsoft Edge（推薦）

Edge 提供最佳的 Windows PWA 集成，具有原生功能。

1. **在 Edge 中打開 OneUptime**
   - 啓動 Microsoft Edge 瀏覽器
   - 導航至您的 OneUptime 實例 URL
   - 登錄您的 OneUptime 賬號
   - 等待頁面完全加載

2. **安裝應用**
   - 查找地址欄中的 **安裝圖標**（⊞）
   - 點擊 **"安裝 OneUptime"** 按鈕
   - 或點擊 **三點菜單** → **應用** → **將此站點安裝爲應用**

3. **自定義安裝**
   - **應用名稱**：根據需要修改（默認：OneUptime）
   - **開始菜單**：選擇是否添加到開始菜單
   - **任務欄**：選擇是否固定到任務欄
   - **桌面**：創建桌面快捷方式

4. **完成安裝**
   - 點擊 **"安裝"** 完成
   - OneUptime 將在自己的窗口中打開
   - 在開始菜單的已安裝應用下找到它

### 方法二：Google Chrome

Chrome 提供出色的 PWA 支持，具有豐富的桌面集成。

1. **在 Chrome 中打開 OneUptime**
   - 啓動 Google Chrome
   - 前往您的 OneUptime 實例
   - 確保已登錄
   - 允許頁面完全加載

2. **通過地址欄安裝**
   - 查找地址欄中的 **安裝圖標**（⊞）
   - 點擊 **"安裝 OneUptime"**
   - 或使用菜單：**三個點** → **更多工具** → **創建快捷方式**

3. **安裝選項**
   - 勾選 **"作爲窗口打開"** 以獲得類應用體驗
   - 根據需要自定義應用名稱
   - 點擊 **"安裝"** 或 **"創建"**

4. **啓動應用**
   - 在 Windows 開始菜單中找到 OneUptime
   - 或從桌面快捷方式啓動
   - 應用在專用窗口中打開

### 方法三：Firefox

Firefox 支持 PWA 安裝，具有基本的桌面集成。

1. **在 Firefox 中打開 OneUptime**
   - 啓動 Firefox 瀏覽器
   - 導航至 OneUptime URL
   - 完成登錄流程

2. **安裝 PWA**
   - 查找 **安裝提示** 或橫幅
   - 或點擊 **菜單** → **安裝**
   - 如果可用，點擊 **"添加到主屏幕"** 等效選項


### 啓動配置
1. **自動啓動**：配置 OneUptime 隨 Windows 啓動
   - 右鍵單擊任務欄 → 任務管理器 → 啓動
   - 根據需要啓用 OneUptime
2. **默認大小**：設置首選窗口大小和位置

### 通知設置
1. **Windows 通知**
   - 設置 → 系統 → 通知和操作
   - 找到 OneUptime 並配置警報偏好
   - 爲事件啓用橫幅通知

2. **專注助手**
   - 配置勿擾設置
   - 允許 OneUptime 緊急通知
   - 爲不同警報類型設置優先級

## 故障排查

### 安裝問題

**安裝按鈕不出現：**
```
解決方案：
1. 確保使用 Edge 或 Chrome（推薦瀏覽器）
2. 驗證與 OneUptime 實例的 HTTPS 連接
3. 清除瀏覽器緩存和 Cookie
4. 將瀏覽器更新到最新版本
5. 檢查服務器上是否滿足 PWA 要求
6. 臨時禁用瀏覽器擴展
```

**安裝失敗或崩潰：**
```
解決方案：
1. 以管理員身份運行瀏覽器
2. 檢查 Windows 用戶賬戶控制（UAC）設置
3. 確保足夠的磁盤空間（最低 100MB）
4. 臨時禁用防病毒軟件
5. 完全清除瀏覽器數據
6. 重啓 Windows 後重試
```

**應用未出現在開始菜單中：**
```
解決方案：
1. 在 Windows 搜索中搜索「OneUptime」
2. 檢查是否以不同名稱安裝
3. 在"最近添加"應用部分查找
4. 重新安裝並確保勾選"添加到開始菜單"
5. 如有必要，手動創建快捷方式
```

### 通知問題

**Windows 通知不工作：**
```
解決方案：
1. Windows 設置 → 系統 → 通知和操作
2. 爲 OneUptime 啓用通知
3. 檢查專注助手設置
4. 確保 OneUptime 中的通知權限
5. 先測試簡單通知
```

## 卸載

### 完全移除
1. **Windows 設置方法**
   - 設置 → 應用 → 應用和功能
   - 搜索「OneUptime」
   - 點擊並選擇"卸載"

2. **瀏覽器方法**
   - 打開 Edge/Chrome
   - 前往 edge://apps/ 或 chrome://apps/
   - 找到 OneUptime
   - 點擊選項 → 卸載

3. **開始菜單方法**
   - 右鍵單擊開始菜單中的 OneUptime
   - 選擇"卸載"
   - 確認移除
