# API 參考

OneUptime 提供全面的 REST API，允許您將監控、事件管理和狀態頁面功能集成到您的應用程序和工作流中。**您在 OneUptime 控制台上能做的任何事情，都可以通過 API 完成，從而實現自動化和自定義集成。**

### 入門

我們的 API 圍繞 REST 原則構建，使用標準 HTTP 響應代碼、認證和請求方法。所有 API 端點均返回 JSON 響應。


### 認證

所有 API 請求都需要使用 API 密鑰進行認證。您可以在 OneUptime 控制台的設置 > API 密鑰下生成 API 密鑰。
