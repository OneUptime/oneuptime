# ドメインモニター

ドメインモニタリングを使用すると、ドメイン名の登録状況と有効期限を監視できます。OneUptimeは定期的にWHOIS検索を実行して、ドメインの正常性を追跡し、有効期限前にアラートを発します。

## 概要

ドメインモニターはドメインのWHOISデータを照会して登録の詳細を追跡します。これにより、以下のことが可能になります。

- ドメインの有効期限日の監視
- 期限切れまたは間もなく期限切れになるドメインの検出
- ドメインレジストラ情報の追跡
- ネームサーバー設定の確認
- ドメインステータスコードの監視

## ドメインモニターの作成

1. OneUptime ダッシュボードで **モニター** を開きます
2. **モニターの作成** をクリックします
3. モニタータイプとして **ドメイン** を選択します
4. 監視するドメイン名を入力します
5. 必要に応じて監視条件を設定します

## 設定オプション

### 基本設定

| フィールド | 説明 | 必須 |
|-------|-------------|----------|
| ドメイン名 | 監視するドメイン（例：`example.com`） | はい |

### 詳細設定

| フィールド | 説明 | デフォルト |
|-------|-------------|---------|
| タイムアウト（ミリ秒） | WHOISレスポンスを待つ時間 | 10000 |
| リトライ回数 | 失敗時の再試行回数 | 3 |

## 監視条件

以下の条件に基づいて、ドメインがオンライン、パフォーマンス低下、またはオフラインと判定されるタイミングを設定できます。

### 利用可能なチェックタイプ

| チェックタイプ | 説明 |
|------------|-------------|
| ドメインの残存日数 | ドメイン登録が有効期限を迎えるまでの日数 |
| ドメインレジストラ | ドメインレジストラの名前 |
| ドメインネームサーバー | ドメインのネームサーバーホスト名 |
| ドメインステータスコード | WHOISのドメインステータスコード |
| ドメインは期限切れか | ドメインの有効期限が切れているか |

### フィルタータイプ

**ドメインは期限切れか** の場合：

- **True** — ドメインの有効期限が切れている
- **False** — ドメインの有効期限が切れていない

**ドメインの残存日数** の場合：

- **より大きい**、**より小さい**、**以上**、**以下**、**等しい**、**等しくない**

**ドメインレジストラ**、**ドメインネームサーバー**、**ドメインステータスコード** の場合：

- **含む** — 値が指定したテキストを含む
- **含まない** — 値が指定したテキストを含まない
- **始まる** — 値が指定したテキストで始まる
- **終わる** — 値が指定したテキストで終わる
- **等しい** — 値が完全一致する
- **等しくない** — 値が一致しない

### 条件の例

#### ドメインが30日以内に有効期限を迎える場合にアラートを発する

- **チェック対象**：ドメインの残存日数
- **フィルタータイプ**：より小さい
- **値**：30

#### ドメインが期限切れの場合にオフラインとしてマークする

- **チェック対象**：ドメインは期限切れか
- **フィルタータイプ**：True

#### ネームサーバーが正しいか確認する

- **チェック対象**：ドメインネームサーバー
- **フィルタータイプ**：含む
- **値**：`ns1.example.com`

## ベストプラクティス

1. **早期警告を設定する** — 有効期限の60日前にパフォーマンス低下アラートを、14日前にオフラインアラートを設定します
2. **すべての重要なドメインを監視する** — プライマリドメイン、個別に登録されたサブドメイン、メールやAPIに使用するドメインを含めます
3. **レジストラの変更を追跡する** — 不正なドメイン移管を検出するためにレジストラフィールドを監視します
