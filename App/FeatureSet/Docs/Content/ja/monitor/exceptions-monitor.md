# 例外モニター

例外モニタリングを使用すると、アプリケーションの例外とエラーを監視し、例外数が設定したしきい値を超えたときにアラートをトリガーできます。OneUptimeはテレメトリーサービスからの例外データを時間ウィンドウで評価します。

## 概要

例外モニターは特定の条件に一致する例外を数えてフィルタリングします。これにより、以下のことが可能になります。

- アプリケーションの例外急増に関するアラート
- 特定の例外タイプの監視
- エラーメッセージによる例外の検索
- 解決済みとアクティブな例外の個別追跡
- エラーパターンからアプリケーションの安定性問題を検出

## 例外モニターの作成

1. OneUptime ダッシュボードで **モニター** を開きます
2. **モニターの作成** をクリックします
3. モニタータイプとして **例外** を選択します
4. 監視するテレメトリーサービスを選択します
5. 必要に応じて例外フィルターと条件を設定します

## 設定オプション

### テレメトリーサービス

例外を監視する1つ以上のサービスを選択します。サービスはOpenTelemetryを介してOneUptimeに例外データを送信している必要があります。

### 例外フィルター

| フィルター           | 説明                                                                    | 必須   |
| -------------------- | ----------------------------------------------------------------------- | ------ |
| 例外タイプ           | 例外タイプ名でフィルタリング（例：`NullPointerException`、`TypeError`） | いいえ |
| メッセージ           | 例外メッセージ内のテキスト検索                                          | いいえ |
| 解決済みを含む       | 解決済みとしてマークされた例外を含める（デフォルト：false）             | いいえ |
| アーカイブ済みを含む | アーカイブされた例外を含める（デフォルト：false）                       | いいえ |
| 時間ウィンドウ       | 例外を検索する過去の期間（秒単位、デフォルト：60）                      | いいえ |

## 監視条件

### 利用可能なチェックタイプ

| チェックタイプ | 説明                                           |
| -------------- | ---------------------------------------------- |
| 例外数         | 時間ウィンドウ内でフィルターに一致する例外の数 |

### フィルタータイプ

- **より大きい** — 例外数がしきい値を超える
- **より小さい** — 例外数がしきい値を下回る
- **以上** — 例外数がしきい値以上
- **以下** — 例外数がしきい値以下
- **等しい** — 例外数が完全一致
- **等しくない** — 例外数が一致しない

### 条件の例

#### 60秒間で10件以上の例外が発生した場合にアラートを発する

- **時間ウィンドウ**：60秒
- **チェック対象**：例外数
- **フィルタータイプ**：より大きい
- **値**：10

#### NullPointerExceptionが1件でも発生した場合にアラートを発する

- **例外タイプ**：`NullPointerException`
- **時間ウィンドウ**：60秒
- **チェック対象**：例外数
- **フィルタータイプ**：より大きい
- **値**：0

#### 特定のメッセージを含む例外を監視する

- **メッセージ**：`out of memory`
- **時間ウィンドウ**：300秒
- **チェック対象**：例外数
- **フィルタータイプ**：より大きい
- **値**：0

## セットアップ要件

例外モニタリングには、OpenTelemetryを介してOneUptimeに例外データを送信するアプリケーションが必要です。セットアップ手順については、[OpenTelemetry](/docs/telemetry/open-telemetry)のドキュメントを参照してください。
