# DNSモニター

DNSモニタリングを使用すると、ドメインのDNS解決の正常性と正確性を監視できます。OneUptimeは定期的にDNSレコードを照会し、設定した条件に基づいてレスポンスを検証します。

## 概要

DNSモニターは特定のレコードタイプのDNSサーバーに問い合わせを行い、結果を評価します。これにより、以下のことが可能になります。

- DNSサービスの可用性の監視
- DNSレコードが正しい値を返していることの確認
- DNS解決のレスポンスタイムの追跡
- DNSSEC設定の検証
- DNS伝播の問題やハイジャックの検出

## DNSモニターの作成

1. OneUptime ダッシュボードで **モニター** を開きます
2. **モニターの作成** をクリックします
3. モニタータイプとして **DNS** を選択します
4. 照会するドメイン名とレコードタイプを入力します
5. 必要に応じて監視条件を設定します

## 設定オプション

### 基本設定

| フィールド | 説明 | 必須 |
|-------|-------------|----------|
| ドメイン名 | 照会するドメイン（例：`example.com`） | はい |
| レコードタイプ | 照会するDNSレコードタイプ | はい |
| DNSサーバー | 使用するカスタムDNSサーバー（例：`8.8.8.8`）。空欄にするとシステムデフォルトを使用 | いいえ |

### サポートされるレコードタイプ

| レコードタイプ | 説明 |
|-------------|-------------|
| A | IPv4アドレスレコード |
| AAAA | IPv6アドレスレコード |
| CNAME | 正規名（エイリアス）レコード |
| MX | メール交換レコード |
| NS | ネームサーバーレコード |
| TXT | テキストレコード（SPF、DKIMなど） |
| SOA | ゾーン起点レコード |
| PTR | ポインターレコード（逆引きDNS） |
| SRV | サービスロケーターレコード |
| CAA | 認証局認可レコード |

### 詳細設定

| フィールド | 説明 | デフォルト |
|-------|-------------|---------|
| ポート | DNSポート番号 | 53 |
| タイムアウト（ミリ秒） | レスポンスを待つ時間 | 5000 |
| リトライ回数 | 失敗時の再試行回数 | 3 |

## 監視条件

以下の条件に基づいて、DNSがオンライン、パフォーマンス低下、またはオフラインと判定されるタイミングを設定できます。

### 利用可能なチェックタイプ

| チェックタイプ | 説明 |
|------------|-------------|
| DNSはオンラインか | DNSサーバーが照会に応答するか |
| DNSレスポンスタイム（ミリ秒） | 照会のレスポンスタイム（ミリ秒） |
| DNSレコードが存在するか | 照会に対してDNSレコードが存在するか |
| DNSレコード値 | DNSレコードが返す値 |
| DNSSECは有効か | DNSSEC検証が通過するか |

### フィルタータイプ

**DNSはオンラインか**、**DNSレコードが存在するか**、**DNSSECは有効か** の場合：

- **True** — 条件が真
- **False** — 条件が偽

**DNSレスポンスタイム** の場合：

- **より大きい**、**より小さい**、**以上**、**以下**、**等しい**、**等しくない**

**DNSレコード値** の場合：

- **含む** — レコード値が指定したテキストを含む
- **含まない** — レコード値が指定したテキストを含まない
- **始まる** — レコード値が指定したテキストで始まる
- **終わる** — レコード値が指定したテキストで終わる
- **等しい** — レコード値が完全一致する
- **等しくない** — レコード値が一致しない

### 条件の例

#### DNSが解決しているか確認する

- **チェック対象**：DNSはオンラインか
- **フィルタータイプ**：True

#### AレコードのIPアドレスが正しいか確認する

- **チェック対象**：DNSレコード値
- **フィルタータイプ**：等しい
- **値**：`93.184.216.34`

#### DNSのレスポンスが遅い場合にアラートを発する

- **チェック対象**：DNSレスポンスタイム（ミリ秒）
- **フィルタータイプ**：より大きい
- **値**：500

#### DNSSECが有効か確認する

- **チェック対象**：DNSSECは有効か
- **フィルタータイプ**：True
