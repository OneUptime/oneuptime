# iOS インストールガイド

外出先でシームレスな監視を行うために、OneUptime を iPhone または iPad にネイティブアプリとしてインストールします。

## インストール方法

### 方法 1: Safari（推奨）

Safari は iOS デバイスで最良の PWA 体験を提供します。

1. **Safari で OneUptime を開く**
   - iOS デバイスで Safari を起動します
   - OneUptime インスタンスの URL に移動します
   - ページが完全に読み込まれるまで待ちます
   - OneUptime アカウントにサインインしていることを確認します

2. **共有メニューにアクセスする**
   - 下部ツールバーの**共有ボタン**（上矢印のある四角）をタップします
   - 共有オプションをスクロールして「ホーム画面に追加」を見つけます

3. **ホーム画面に追加する**
   - **「ホーム画面に追加」**をタップします
   - アプリ名をカスタマイズします（デフォルト: "OneUptime"）
   - 右上の**「追加」**をタップします

4. **アプリを起動する**
   - ホーム画面で OneUptime アイコンを見つけます
   - タップしてフルスクリーンアプリモードで起動します

### 方法 2: Chrome ブラウザ

Chrome でも動作しますが、最良の iOS PWA 体験には Safari が推奨されます。

1. **Chrome で OneUptime を開く**
   - Chrome ブラウザを起動します
   - OneUptime インスタンスに移動します
   - ページが完全に読み込まれるまで待ちます

2. **ホーム画面に追加する**
   - **三点メニュー**（その他のオプション）をタップします
   - **「ホーム画面に追加」**を選択します
   - 必要に応じてアプリ名をカスタマイズします
   - **「追加」**をタップします

### 方法 3: その他のブラウザ

Firefox、Edge、その他のブラウザは基本的な PWA インストールをサポートしています。

1. **OneUptime を開く**
   - 使用するブラウザを起動します
   - OneUptime の URL に移動します
   - ページが完全に読み込まれるまで待ちます

2. **インストールオプションを探す**
   - ブラウザメニューで「ホーム画面に追加」または「インストール」を確認します
   - ブラウザ固有のインストールプロンプトに従います

### カスタマイズオプション

### アプリアイコンと名前
- **カスタム名**: インストール中または後で変更できます
- **アイコンの配置**: フォルダーまたは特定のホーム画面ページに整理できます
- **バッジ通知**: 未読のインシデント数を表示します

### 通知設定
1. **通知を有効にする**
   - プロンプトが表示されたら**「許可」**をタップします
   - または設定 → 通知 → OneUptime に移動します
   - 包括的な監視のためすべての通知タイプを有効にします

2. **アラートスタイルをカスタマイズする**
   - **ロック画面**: ロックされたデバイスにインシデントアラートを表示します
   - **バナースタイル**: 一時的または固定バナーを選択します
   - **サウンド**: 通知サウンドとバイブレーションをカスタマイズします
   - **重要なアラート**: 高優先度のインシデントに対して有効にします（権限が必要）

## トラブルシューティング

### インストールの問題

**「ホーム画面に追加」が表示されない場合:**
```
解決策:
1. Safari を使用していることを確認します（最良の互換性）
2. ページを更新して 30 秒待ちます
3. 正しい OneUptime URL を開いているか確認します
4. HTTPS 接続を確認します（鍵アイコンを確認）
5. Safari キャッシュをクリアします: 設定 → Safari → 履歴と Web サイトデータを消去
```

**インストールは完了するがアイコンが表示されない場合:**
```
解決策:
1. すべてのホーム画面ページを確認します
2. App ライブラリを確認します（最後のホーム画面の左にスワイプ）
3. Spotlight 検索で「OneUptime」を検索します
4. デバイスを再起動して再確認します
5. 必要に応じて再インストールします
```

**起動時にアプリがクラッシュする場合:**
```
解決策:
1. アプリを強制終了して再度開きます
2. iOS デバイスを再起動します
3. Safari キャッシュをクリアして再インストールします
4. iOS バージョンが 11.3 以降であることを確認します
5. デバイスのストレージ容量を確保します
```

### 通知の問題

**プッシュ通知が届かない場合:**
```
以下の設定を確認してください:
1. 設定 → 通知 → OneUptime → 通知を許可
2. 設定 → スクリーンタイム → コンテンツとプライバシーの制限 → 許可されたアプリ
3. 「マナーモード」設定
4. OneUptime ダッシュボードで通知設定を確認
5. OneUptime からサインアウトして再サインインします
```

**通知が遅延または届かない場合:**
```
解決策:
1. アプリをバックグラウンドで実行したままにします（強制終了しない）
2. 重要な監視中は低電力モードを無効にします
3. バックグラウンドアプリの更新を確認: 設定 → 一般 → バックグラウンドアプリの更新
4. 十分なストレージ容量があることを確認します
```

## アンインストール

### ホーム画面から削除する
1. OneUptime アプリアイコンを**長押し**します
2. **「アプリを削除」**をタップします
3. **「アプリを削除」**を選択します
4. 削除を確認します

### 別の方法
1. **設定 → 一般 → iPhone ストレージ**に移動します
2. アプリリストで**OneUptime**を見つけます
3. **「アプリを削除」**をタップします
4. 削除を確認します

## 更新とメンテナンス

### 自動更新
- OneUptime PWA はオンライン時に自動的に更新されます
- App Store の更新は不要です
- サーバーデプロイ後すぐに新機能が利用可能になります
- 重要なセキュリティ更新が即時に適用されます

## iPad 固有のインストール

### iPad の向上した体験
1. **大きなインターフェース**: iPad の画面サイズに最適化されたレイアウト
2. **マルチウィンドウ**: 複数の OneUptime ウィンドウを同時に実行
3. **キーボードショートカット**: 外部キーボードの完全サポート
4. **ドラッグ＆ドロップ**: OneUptime と他のアプリ間でデータを移動

### iPad のインストール手順
iPhone のインストールと同様ですが、追加の考慮事項があります:
- 最適なダッシュボード表示のために横向きモードを使用します
- 他の生産性アプリとの Split View の設定を検討します
- 一般的なアクションのキーボードショートカットを設定します

## Apple Watch との統合

OneUptime には専用の watchOS アプリはありませんが、以下が可能です:
- **通知の受信**: インシデントアラートが Apple Watch に表示されます
- **クイックアクション**: Watch の通知からインシデントを承認します
- **Siri 統合**: システムステータスを Siri に尋ねます（設定時）

## ベストプラクティス

### セキュリティの推奨事項
1. **URL の確認**: 組織の公式 OneUptime インスタンスからのみインストールします
2. **HTTPS のみ**: セキュアな接続を確保します（鍵アイコンを確認）
3. **定期的な更新**: セキュリティパッチのために iOS を最新の状態に保ちます
4. **アプリの権限**: 必要な権限のみを付与します

### パフォーマンスの最適化
1. **Wi-Fi インストール**: 初期インストールと大きな更新には Wi-Fi を使用します
2. **バックグラウンド更新**: タイムリーな通知のために有効にします
3. **ストレージ管理**: 十分な空き容量を確保します
4. **定期的な再起動**: 最適なパフォーマンスのためにアプリを週に一度再起動します
