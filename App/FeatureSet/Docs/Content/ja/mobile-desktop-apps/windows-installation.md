# Windows インストールガイド

包括的な監視とインシデント管理のために、OneUptime を Windows にデスクトップアプリケーションとしてインストールします。

## インストール方法

### 方法 1: Microsoft Edge（推奨）

Edge は Windows PWA 統合とネイティブ機能において最良の体験を提供します。

1. **Edge で OneUptime を開く**

   - Microsoft Edge ブラウザを起動します
   - OneUptime インスタンスの URL に移動します
   - OneUptime アカウントにサインインします
   - ページが完全に読み込まれるまで待ちます

2. **アプリをインストールする**

   - アドレスバーの**インストールアイコン**（⊞）を探します
   - **「OneUptime をインストール」**ボタンをクリックします
   - または**三点メニュー** → **アプリ** → **このサイトをアプリとしてインストール**をクリックします

3. **インストールをカスタマイズする**

   - **アプリ名**: 必要に応じて変更します（デフォルト: OneUptime）
   - **スタートメニュー**: スタートメニューに追加するか選択します
   - **タスクバー**: タスクバーにピン留めするオプション
   - **デスクトップ**: デスクトップショートカットを作成します

4. **インストールを完了する**
   - **「インストール」**をクリックして完了します
   - OneUptime が独自のウィンドウで開きます
   - インストール済みアプリのスタートメニューで見つかります

### 方法 2: Google Chrome

Chrome は豊富なデスクトップ統合を備えた優れた PWA サポートを提供します。

1. **Chrome で OneUptime を開く**

   - Google Chrome を起動します
   - OneUptime インスタンスに移動します
   - サインインしていることを確認します
   - ページが完全に読み込まれるまで待ちます

2. **アドレスバーからインストールする**

   - アドレスバーの**インストールアイコン**（⊞）を探します
   - **「OneUptime をインストール」**をクリックします
   - またはメニューを使用: **三点** → **その他のツール** → **ショートカットを作成**

3. **インストールオプション**

   - アプリのような体験のために**「ウィンドウで開く」**をチェックします
   - 必要に応じてアプリ名をカスタマイズします
   - **「インストール」**または**「作成」**をクリックします

4. **アプリを起動する**
   - Windows スタートメニューで OneUptime を見つけます
   - またはデスクトップショートカットから起動します
   - アプリが専用のウィンドウで開きます

### 方法 3: Firefox

Firefox は基本的なデスクトップ統合を備えた PWA インストールをサポートしています。

1. **Firefox で OneUptime を開く**

   - Firefox ブラウザを起動します
   - OneUptime の URL に移動します
   - サインインプロセスを完了します

2. **PWA をインストールする**
   - **インストールプロンプト**またはバナーを探します
   - または**メニュー** → **インストール**をクリックします
   - 利用可能な場合は**「ホーム画面に追加」**相当をクリックします

### スタートアップ設定

1. **自動起動**: Windows 起動時に OneUptime が起動するように設定します
   - タスクバーを右クリック → タスクマネージャー → スタートアップ
   - 必要に応じて OneUptime を有効にします
2. **デフォルトサイズ**: 好みのウィンドウサイズと位置を設定します

### 通知設定

1. **Windows 通知**

   - 設定 → システム → 通知とアクション
   - OneUptime を見つけてアラート設定を構成します
   - インシデントのバナー通知を有効にします

2. **フォーカスアシスト**
   - 「マナーモード」設定を構成します
   - OneUptime の重要な通知を許可します
   - 異なるアラートタイプの優先度レベルを設定します

## トラブルシューティング

### インストールの問題

**インストールボタンが表示されない場合:**

```
解決策:
1. Edge または Chrome を使用していることを確認します（推奨ブラウザ）
2. OneUptime インスタンスへの HTTPS 接続を確認します
3. ブラウザのキャッシュとクッキーをクリアします
4. ブラウザを最新バージョンに更新します
5. サーバーで PWA 要件が満たされているか確認します
6. ブラウザ拡張機能を一時的に無効にします
```

**インストールが失敗またはクラッシュする場合:**

```
解決策:
1. 管理者としてブラウザを実行します
2. Windows ユーザーアカウント制御（UAC）設定を確認します
3. 十分なディスク容量があることを確認します（最小 100MB）
4. ウイルス対策ソフトを一時的に無効にします
5. ブラウザデータを完全にクリアします
6. Windows を再起動して再試行します
```

**アプリがスタートメニューに表示されない場合:**

```
解決策:
1. Windows 検索で「OneUptime」を検索します
2. 別の名前でインストールされていないか確認します
3. 「最近追加したもの」アプリセクションを確認します
4. 「スタートメニューに追加」がチェックされていることを確認して再インストールします
5. 必要に応じて手動でショートカットを作成します
```

### 通知の問題

**Windows 通知が機能しない場合:**

```
解決策:
1. Windows 設定 → システム → 通知とアクション
2. OneUptime の通知を有効にします
3. フォーカスアシスト設定を確認します
4. OneUptime での通知権限を確認します
5. まずシンプルな通知でテストします
```

## アンインストール

### 完全な削除

1. **Windows 設定の方法**

   - 設定 → アプリ → アプリと機能
   - 「OneUptime」を検索します
   - クリックして「アンインストール」を選択します

2. **ブラウザの方法**

   - Edge/Chrome を開きます
   - edge://apps/ または chrome://apps/ に移動します
   - OneUptime を見つけます
   - オプション → アンインストールをクリックします

3. **スタートメニューの方法**
   - スタートメニューで OneUptime を右クリックします
   - 「アンインストール」を選択します
   - 削除を確認します

## 更新とメンテナンス

### 自動更新

- OneUptime PWA はオンライン時に自動的に更新されます
- 手動の操作は不要です
- 再起動時に即座に更新が適用されます
- 重要なパッチが即時にデプロイされます
