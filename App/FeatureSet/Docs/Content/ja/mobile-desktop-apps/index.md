# OneUptime モバイルおよびデスクトップアプリ

OneUptime は、プログレッシブウェブアプリ（PWA）技術を通じて、ネイティブに近いモバイルおよびデスクトップアプリケーションを提供しています。これらのアプリは、アプリストアからのダウンロードを必要とせずに、すべての主要プラットフォームでシームレスな体験を提供します。

## プログレッシブウェブアプリ（PWA）とは？

プログレッシブウェブアプリは、モダンなウェブ機能を使用してユーザーにアプリのような体験を提供するウェブアプリケーションです。

## はじめに

1. サポートされているブラウザで OneUptime インスタンスにアクセスします
2. 「インストール」または「ホーム画面に追加」プロンプトを探します
3. プラットフォーム固有のインストール手順に従います
4. デバイスのアプリドロワーまたはデスクトップからアプリを起動します

## トラブルシューティング

### 一般的な問題

**インストールオプションが表示されない場合:**
- サポートされているブラウザを使用していることを確認します
- OneUptime インスタンスが HTTPS で提供されていることを確認します
- ページを更新するかブラウザキャッシュをクリアしてみます

**プッシュ通知が機能しない場合:**
- プロンプトが表示されたら通知権限を付与します
- ブラウザの通知設定を確認します
- OneUptime インスタンスで通知が設定されていることを確認します


## サポート

OneUptime PWA で問題が発生した場合:

1. 上記のトラブルシューティングセクションを確認します
2. ブラウザが最小要件を満たしていることを確認します
3. OneUptime ダッシュボードからサポートに連絡します
4. [GitHub リポジトリ](https://github.com/OneUptime/oneuptime)で問題を報告します
