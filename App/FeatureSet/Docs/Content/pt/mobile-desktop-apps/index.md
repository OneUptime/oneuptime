# Aplicativos Móveis e Desktop do OneUptime

O OneUptime oferece aplicativos móveis e desktop com aparência nativa por meio da tecnologia Progressive Web App (PWA). Esses aplicativos fornecem uma experiência perfeita em todas as principais plataformas sem exigir downloads de lojas de aplicativos.

## O que é um Progressive Web App (PWA)?

Um Progressive Web App é um aplicativo web que usa recursos web modernos para oferecer uma experiência semelhante a um aplicativo aos usuários.

## Primeiros Passos

1. Visite sua instância do OneUptime em um navegador suportado
2. Procure o prompt "Instalar" ou "Adicionar à Tela Inicial"
3. Siga as etapas de instalação específicas da plataforma
4. Inicie o aplicativo a partir da gaveta de aplicativos ou área de trabalho do seu dispositivo

## Solução de Problemas

### Problemas Comuns

**Opção de instalação não aparecendo:**
- Certifique-se de estar usando um navegador suportado
- Verifique se sua instância do OneUptime é servida via HTTPS
- Tente atualizar a página ou limpar o cache do navegador

**Notificações push não estão funcionando:**
- Conceda permissões de notificação quando solicitado
- Verifique as configurações de notificação do navegador
- Certifique-se de que a instância do OneUptime tenha notificações configuradas


## Suporte

Se você encontrar algum problema com o PWA do OneUptime:

1. Verifique a seção de solução de problemas acima
2. Verifique se seu navegador atende aos requisitos mínimos
3. Entre em contato com o suporte pelo seu painel do OneUptime
4. Relate problemas em nosso [repositório GitHub](https://github.com/OneUptime/oneuptime)
