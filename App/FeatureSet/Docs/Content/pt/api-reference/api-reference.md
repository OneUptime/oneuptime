# Referência de API

O OneUptime fornece uma API REST abrangente que permite integrar monitoramento, gerenciamento de incidentes e funcionalidade de página de status em seus aplicativos e fluxos de trabalho. **Tudo o que você pode fazer no painel do OneUptime também pode ser feito através da API, possibilitando automação e integrações personalizadas.**

### Primeiros Passos

Nossa API é organizada em torno dos princípios REST e usa códigos de resposta HTTP padrão, autenticação e verbos. Todos os endpoints de API retornam respostas JSON.


### Autenticação

Todas as requisições de API requerem autenticação usando chaves de API. Você pode gerar chaves de API no seu painel do OneUptime em Configurações > Chaves de API.
