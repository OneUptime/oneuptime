# Белый список IP-адресов для OneUptime.com

Если вы используете OneUptime.com и хотите добавить наши IP-адреса в белый список из соображений безопасности, вы можете сделать это, следуя приведённым ниже инструкциям.

Пожалуйста, добавьте следующие IP-адреса в белый список вашего брандмауэра, чтобы oneuptime.com мог обращаться к вашим ресурсам.

{{IP_WHITELIST}}

Эти IP-адреса могут изменяться; мы заблаговременно уведомим вас, если это произойдёт.

## Программное получение IP-адресов

Вы также можете программно получить список исходящих IP-адресов зондов через следующую конечную точку API:

```
GET https://oneuptime.com/ip-whitelist
```

Возвращает JSON-ответ:

```json
{
  "ipWhitelist": ["<list of IPs>"]
}
```

Вы можете использовать эту конечную точку для автоматического обновления белого списка брандмауэра.
