# CLI OneUptime

CLI OneUptime — это интерфейс командной строки для управления ресурсами OneUptime непосредственно из терминала. Поддерживает полные CRUD-операции с мониторами, инцидентами, алертами, страницами статуса и многим другим.

## Возможности

- **Поддержка нескольких сред** с именованными контекстами для production, staging и development
- **Автоматическое обнаружение** доступных ресурсов из вашего экземпляра OneUptime
- **Гибкая аутентификация** через флаги CLI, переменные окружения или сохранённые контексты
- **Умное форматирование вывода** с режимами отображения JSON, таблицы и расширенного формата
- **Поддержка скриптов** для конвейеров CI/CD и рабочих процессов автоматизации

## Установка

```bash
npm install -g @oneuptime/cli
```

## Быстрый старт

```bash
# Аутентификация в вашем экземпляре OneUptime
oneuptime login <your-api-key> https://oneuptime.com

# Список ваших мониторов
oneuptime monitor list

# Просмотр конкретного инцидента
oneuptime incident get <incident-id>

# Просмотр всех доступных ресурсов
oneuptime resources
```

## Глобальные опции

Эти флаги можно использовать с любой командой:

| Флаг                    | Описание                                       |
| ----------------------- | ---------------------------------------------- |
| `--api-key <key>`       | Переопределить ключ API для этой команды       |
| `--url <url>`           | Переопределить URL экземпляра для этой команды |
| `--context <name>`      | Использовать конкретный именованный контекст   |
| `-o, --output <format>` | Формат вывода: `json`, `table`, `wide`         |
| `--no-color`            | Отключить цветной вывод                        |
| `--help`                | Показать справку по команде                    |
| `--version`             | Показать версию CLI                            |

## Получение справки

```bash
# Общая справка
oneuptime --help

# Справка по конкретной команде
oneuptime monitor --help
oneuptime monitor list --help
```
