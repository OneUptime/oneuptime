# Подключение OneUptime к Slack

### Шаги по подключению OneUptime к Slack

1. **Создайте учётную запись OneUptime**
    - Перейдите на [OneUptime.com](https://oneuptime.com) и создайте учётную запись.
    - После создания учётной записи создайте новый проект.

2. **Подключите Slack к проекту OneUptime**
    - В вашем проекте OneUptime перейдите в **Настройки проекта** > **Slack**.
    - Следуйте инструкциям для подключения учётной записи Slack к проекту OneUptime.

3. **Настройте уведомления об инцидентах**
    - После подключения учётной записи Slack перейдите на **Страницу инцидентов** > **Slack**.
    - Добавьте правила для отправки уведомлений об инцидентах в Slack. Например, можно создать правило, которое создаёт новый канал Slack и приглашает ответственных за инцидент при его создании.

4. **Настройте уведомления об оповещениях и плановых работах**
    - Аналогичные правила можно применить к оповещениям и плановым работам, перейдя на соответствующие страницы и настроив нужные правила.
