# Монитор доменов

Мониторинг доменов позволяет отслеживать статус регистрации и срок истечения действия доменных имён. OneUptime периодически выполняет WHOIS-запросы для отслеживания работоспособности домена и предупреждения об истечении срока регистрации.

## Обзор

Мониторы доменов запрашивают WHOIS-данные для отслеживания сведений о регистрации доменов. Это позволяет:

- Мониторинг сроков истечения регистрации доменов
- Обнаружение истёкших или скоро истекающих доменов
- Отслеживание информации о регистраторе домена
- Проверка конфигурации серверов имён
- Мониторинг кодов статуса доменов

## Создание монитора домена

1. Перейдите в раздел **Мониторы** на панели управления OneUptime
2. Нажмите **Создать монитор**
3. Выберите тип монитора **Домен**
4. Введите имя домена для мониторинга
5. При необходимости настройте критерии мониторинга

## Параметры конфигурации

### Основные настройки

| Поле | Описание | Обязательно |
|------|----------|-------------|
| Имя домена | Домен для мониторинга (например, `example.com`) | Да |

### Дополнительные настройки

| Поле | Описание | По умолчанию |
|------|----------|--------------|
| Тайм-аут (мс) | Время ожидания ответа WHOIS | 10000 |
| Повторные попытки | Количество повторных попыток при сбое | 3 |

## Критерии мониторинга

Можно настроить критерии, определяющие, считается ли домен доступным, деградированным или недоступным, на основе:

### Доступные типы проверок

| Тип проверки | Описание |
|--------------|----------|
| Срок истечения домена (дней) | Количество дней до истечения регистрации домена |
| Регистратор домена | Название регистратора домена |
| Сервер имён домена | Имена хостов серверов имён домена |
| Код статуса домена | Коды статуса домена в WHOIS |
| Домен истёк | Истекла ли регистрация домена |

### Типы фильтров

Для **Домен истёк**:

- **True** — регистрация домена истекла
- **False** — регистрация домена не истекла

Для **Срок истечения домена (дней)**:

- **Больше чем**, **Меньше чем**, **Больше или равно**, **Меньше или равно**, **Равно**, **Не равно**

Для **Регистратора домена**, **Сервера имён домена** и **Кода статуса домена**:

- **Содержит** — значение содержит указанный текст
- **Не содержит** — значение не содержит указанный текст
- **Начинается с** — значение начинается с указанного текста
- **Заканчивается на** — значение заканчивается на указанный текст
- **Равно** — значение точно совпадает
- **Не равно** — значение не совпадает

### Примеры критериев

#### Оповещение при истечении домена через 30 дней

- **Проверять**: Срок истечения домена (дней)
- **Тип фильтра**: Меньше чем
- **Значение**: 30

#### Статус «Недоступен» при истечении домена

- **Проверять**: Домен истёк
- **Тип фильтра**: True

#### Проверка корректности серверов имён

- **Проверять**: Сервер имён домена
- **Тип фильтра**: Содержит
- **Значение**: `ns1.example.com`

## Рекомендации по использованию

1. **Устанавливайте ранние предупреждения** — настройте статус деградации за 60 дней и статус недоступности за 14 дней до истечения срока
2. **Отслеживайте все критически важные домены** — включайте основные домены, отдельно зарегистрированные субдомены и любые домены, используемые для электронной почты или API
3. **Отслеживайте изменения регистратора** — мониторинг поля регистратора поможет обнаружить несанкционированные передачи домена
