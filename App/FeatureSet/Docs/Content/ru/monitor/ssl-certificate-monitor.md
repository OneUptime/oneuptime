# Монитор SSL-сертификатов

Мониторинг SSL-сертификатов позволяет отслеживать действительность и срок истечения SSL/TLS-сертификатов на ваших сайтах и сервисах. OneUptime периодически проверяет сертификаты и оповещает вас до истечения их срока действия или при обнаружении других проблем.

## Обзор

Мониторы SSL-сертификатов подключаются к вашим HTTPS-конечным точкам и проверяют SSL/TLS-сертификат. Это позволяет:

- Мониторинг сроков истечения сертификатов
- Обнаружение истёкших или скоро истекающих сертификатов
- Выявление самоподписанных сертификатов
- Проверка действительности сертификатов
- Предотвращение простоев сервисов из-за истёкших сертификатов

## Создание монитора SSL-сертификатов

1. Перейдите в раздел **Мониторы** на панели управления OneUptime
2. Нажмите **Создать монитор**
3. Выберите тип монитора **SSL-сертификат**
4. Введите URL HTTPS-конечной точки для проверки
5. При необходимости настройте критерии мониторинга

## Параметры конфигурации

### URL

Введите полный HTTPS-URL конечной точки, SSL-сертификат которой нужно отслеживать (например, `https://example.com` или `https://example.com:8443`).

## Критерии мониторинга

Можно настроить критерии, определяющие, считается ли статус сертификата допустимым, деградированным или недопустимым, на основе:

### Доступные типы проверок

| Тип проверки                | Описание                                                                |
| --------------------------- | ----------------------------------------------------------------------- |
| Доступен                    | Доступен ли сервер                                                      |
| Действительный сертификат   | Является ли сертификат действительным (не истёкшим, не самоподписанным) |
| Самоподписанный сертификат  | Является ли сертификат самоподписанным                                  |
| Истёкший сертификат         | Истёк ли срок действия сертификата                                      |
| Недействительный сертификат | Является ли сертификат недействительным                                 |
| Истекает через (часов)      | Количество часов до истечения срока сертификата                         |
| Истекает через (дней)       | Количество дней до истечения срока сертификата                          |
| Тайм-аут запроса            | Истёк ли тайм-аут соединения                                            |

### Типы фильтров

Для **Доступен**, **Действительный сертификат**, **Самоподписанный сертификат**, **Истёкший сертификат**, **Недействительный сертификат** и **Тайм-аут запроса**:

- **True** — условие истинно
- **False** — условие ложно

Для **Истекает через (часов)** и **Истекает через (дней)**:

- **Больше чем** — срок истечения более чем через указанное значение
- **Меньше чем** — срок истечения менее чем через указанное значение
- **Больше или равно** — срок истечения через указанное значение или более
- **Меньше или равно** — срок истечения через указанное значение или менее
- **Равно** — срок истечения точно совпадает
- **Не равно** — срок истечения не совпадает

### Примеры критериев

#### Статус «Деградирован» при истечении сертификата через 30 дней

- **Проверять**: Истекает через (дней)
- **Тип фильтра**: Меньше чем
- **Значение**: 30

#### Статус «Недоступен» при истёкшем сертификате

- **Проверять**: Истёкший сертификат
- **Тип фильтра**: True

#### Оповещение при самоподписанном сертификате

- **Проверять**: Самоподписанный сертификат
- **Тип фильтра**: True

#### Статус «Недоступен» при недействительном сертификате

- **Проверять**: Недействительный сертификат
- **Тип фильтра**: True

## Рекомендации по использованию

1. **Устанавливайте несколько порогов** — используйте статус деградации за 30 дней и статус недоступности за 7 дней до истечения срока, чтобы иметь время на обновление
2. **Отслеживайте все конечные точки** — если у вас несколько доменов или поддоменов, создайте монитор для каждого
3. **Включайте нестандартные порты** — не забывайте о сервисах, работающих по HTTPS на нестандартных портах
4. **Мониторинг после обновления** — после обновления сертификата убедитесь, что монитор подтверждает его действительность
