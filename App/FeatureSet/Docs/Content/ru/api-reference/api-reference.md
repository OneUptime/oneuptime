# Справочник по API

OneUptime предоставляет исчерпывающий REST API, позволяющий интегрировать функции мониторинга, управления инцидентами и страниц статуса в ваши приложения и рабочие процессы. **Всё, что можно сделать на панели управления OneUptime, также можно сделать через API, обеспечивая автоматизацию и пользовательские интеграции.**

### Начало работы

Наш API построен на принципах REST и использует стандартные HTTP-коды ответов, аутентификацию и методы. Все конечные точки API возвращают JSON-ответы.


### Аутентификация

Все запросы к API требуют аутентификации с помощью ключей API. Вы можете создать ключи API в панели управления OneUptime в разделе Settings > API Keys.
