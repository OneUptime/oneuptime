# Руководство по установке на iOS

Установите OneUptime как нативное приложение на iPhone или iPad для удобного мониторинга в любом месте.

## Способы установки

### Способ 1: Safari (рекомендуется)

Safari обеспечивает лучший опыт использования PWA на устройствах iOS.

1. **Откройте OneUptime в Safari**
   - Запустите Safari на устройстве iOS
   - Перейдите по URL вашего экземпляра OneUptime
   - Дождитесь полной загрузки страницы
   - Убедитесь, что вы вошли в свою учётную запись OneUptime

2. **Откройте меню «Поделиться»**
   - Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх) на нижней панели инструментов
   - Прокрутите варианты обмена и найдите «На экран «Домой»»

3. **Добавьте на экран «Домой»**
   - Нажмите **«На экран «Домой»»**
   - Задайте название приложения (по умолчанию: «OneUptime»)
   - Нажмите **«Добавить»** в правом верхнем углу

4. **Запустите приложение**
   - Найдите значок OneUptime на экране «Домой»
   - Нажмите, чтобы запустить в полноэкранном режиме приложения

### Способ 2: Браузер Chrome

Chrome работает, однако для лучшего опыта PWA на iOS рекомендуется Safari.

1. **Откройте OneUptime в Chrome**
   - Запустите браузер Chrome
   - Перейдите на ваш экземпляр OneUptime
   - Дождитесь полной загрузки страницы

2. **Добавьте на экран «Домой»**
   - Нажмите **меню с тремя точками** (дополнительные параметры)
   - Выберите **«На экран «Домой»»**
   - При желании задайте имя приложения
   - Нажмите **«Добавить»**

### Способ 3: Другие браузеры

Firefox, Edge и другие браузеры поддерживают базовую установку PWA:

1. **Откройте OneUptime**
   - Запустите предпочитаемый браузер
   - Перейдите по URL OneUptime
   - Дождитесь полной загрузки страницы

2. **Найдите опцию установки**
   - Проверьте меню браузера на наличие «На экран «Домой»» или «Установить»
   - Следуйте инструкциям по установке, характерным для данного браузера

### Параметры настройки

### Значок и название приложения
- **Пользовательское название**: задайте при установке или измените позже
- **Размещение значка**: упорядочивайте в папках или на конкретных страницах экрана «Домой»
- **Значки уведомлений**: отображают количество непрочитанных инцидентов

### Настройка уведомлений
1. **Включите уведомления**
   - При появлении запроса нажмите **«Разрешить»**
   - Или перейдите в Настройки → Уведомления → OneUptime
   - Включите все типы уведомлений для полноценного мониторинга

2. **Настройте стиль оповещений**
   - **Экран блокировки**: отображение оповещений об инцидентах на заблокированном устройстве
   - **Стиль баннера**: выберите временные или постоянные баннеры
   - **Звуки**: настройте звуки и вибрацию уведомлений
   - **Критические оповещения**: включите для высокоприоритетных инцидентов (требуется разрешение)

## Устранение неполадок

### Проблемы с установкой

**«На экран «Домой»» не отображается:**
```
Решения:
1. Убедитесь, что используете Safari (лучшая совместимость)
2. Обновите страницу и подождите 30 секунд
3. Проверьте правильность URL OneUptime
4. Проверьте подключение по HTTPS (наличие значка замка)
5. Очистите кеш Safari: Настройки → Safari → Очистить историю и данные сайтов
```

**Установка завершена, но значок не появился:**
```
Решения:
1. Проверьте все страницы экрана «Домой»
2. Найдите в библиотеке приложений (проведите влево за последней страницей экрана «Домой»)
3. Воспользуйтесь поиском Spotlight для поиска «OneUptime»
4. Перезагрузите устройство и повторите попытку
5. Если необходимо, переустановите приложение
```

**Приложение вылетает при запуске:**
```
Решения:
1. Принудительно закройте и снова откройте приложение
2. Перезагрузите устройство iOS
3. Очистите кеш Safari и переустановите
4. Убедитесь, что версия iOS не ниже 11.3
5. Освободите место в хранилище устройства
```

### Проблемы с уведомлениями

**Не получаю push-уведомления:**
```
Проверьте следующие настройки:
1. Настройки → Уведомления → OneUptime → Разрешить уведомления
2. Настройки → Экранное время → Контент и конфиденциальность → Разрешённые приложения
3. Настройки режима «Не беспокоить»
4. Проверьте настройки уведомлений в панели управления OneUptime
5. Выйдите и снова войдите в OneUptime
```

**Задержанные или пропущенные уведомления:**
```
Решения:
1. Держите приложение запущенным в фоне (не закрывайте принудительно)
2. Отключите режим экономии энергии во время критического мониторинга
3. Проверьте фоновое обновление: Настройки → Основные → Фоновое обновление
4. Убедитесь, что достаточно свободного места в хранилище
```

## Удаление

### Удаление с экрана «Домой»
1. **Удерживайте** значок приложения OneUptime
2. Нажмите **«Удалить приложение»**
3. Выберите **«Удалить»**
4. Подтвердите удаление

### Альтернативный способ
1. Перейдите в **Настройки → Основные → Хранилище iPhone**
2. Найдите **OneUptime** в списке приложений
3. Нажмите **«Удалить приложение»**
4. Подтвердите удаление

## Обновления и техническое обслуживание

## Установка на iPad

### Расширенные возможности iPad
1. **Увеличенный интерфейс**: оптимизированные макеты для экранов iPad
2. **Многооконный режим**: запуск нескольких окон OneUptime одновременно
3. **Сочетания клавиш**: полная поддержка внешних клавиатур
4. **Перетаскивание**: перемещение данных между OneUptime и другими приложениями

### Шаги по установке на iPad
Аналогичны установке на iPhone, с дополнительными особенностями:
- Используйте альбомную ориентацию для оптимального просмотра панелей управления
- Рассмотрите настройку режима Split View с другими рабочими приложениями
- Настройте сочетания клавиш для частых действий

## Интеграция с Apple Watch

Хотя у OneUptime нет отдельного приложения для watchOS, вы можете:
- **Получать уведомления**: оповещения об инцидентах отображаются на Apple Watch
- **Быстрые действия**: подтверждение инцидентов из уведомлений на часах
- **Интеграция с Siri**: запрос статуса системы через Siri (при настройке)

## Расширенная настройка

### Интеграция с приложением «Команды»
Создайте пользовательские команды Siri для OneUptime:
1. Откройте приложение **Команды**
2. Создайте **Новую команду**
3. Добавьте действие **«Открыть приложение»**
4. Выберите **OneUptime**
5. Добавьте голосовую фразу, например «Проверить статус системы»

### Режимы фокусировки
Интегрируйте OneUptime с режимами фокусировки iOS:
1. **Настройки → Фокусирование**
2. Выберите или создайте режим фокусировки
3. **Приложения → Добавить приложения → OneUptime**
4. Настройте поведение уведомлений для разных режимов фокусировки

### Управление экранным временем
Настройте соответствующие параметры экранного времени:
1. **Настройки → Экранное время → Лимиты приложений**
2. Добавьте OneUptime в категорию «Производительность»
3. При необходимости установите соответствующие ограничения использования

## Обновления и техническое обслуживание

### Автоматические обновления
- OneUptime PWA обновляется автоматически при наличии подключения к интернету
- Обновления из App Store не требуются
- Новые функции становятся доступны сразу после развёртывания на сервере
- Критические обновления безопасности применяются мгновенно

### Ручное обновление
Принудительно обновите приложение:
1. Откройте OneUptime в Safari
2. Потяните вниз для обновления
3. Приложение загрузит последнюю версию
4. Новые функции станут доступны немедленно

### Управление кешем
Поддерживайте оптимальную работу приложения:
- **Регулярное использование**: открывайте приложение ежедневно для актуальности кеша
- **Мониторинг хранилища**: поддерживайте не менее 1 ГБ свободного места
- **Доступ к сети**: регулярно подключайтесь к Wi-Fi для получения обновлений

## Рекомендации по использованию

### Рекомендации по безопасности
1. **Проверяйте URL**: устанавливайте только с официального экземпляра OneUptime вашей организации
2. **Только HTTPS**: убедитесь в наличии защищённого соединения (значок замка)
3. **Регулярные обновления**: обновляйте iOS для получения исправлений безопасности
4. **Разрешения приложения**: предоставляйте только необходимые разрешения

### Оптимизация производительности
1. **Установка по Wi-Fi**: используйте Wi-Fi для первоначальной установки и крупных обновлений
2. **Фоновое обновление**: включите для своевременных уведомлений
3. **Управление хранилищем**: поддерживайте достаточно свободного места
4. **Регулярный перезапуск**: перезапускайте приложение еженедельно для оптимальной работы

### Рекомендации по мониторингу
1. **Критические уведомления**: включайте только для высокоприоритетных оповещений
2. **Несколько устройств**: устанавливайте как на iPhone, так и на iPad для резервирования
3. **Командный доступ**: делитесь инструкцией по установке с членами команды
4. **Тестирование**: регулярно проверяйте доставку уведомлений и работу в автономном режиме
