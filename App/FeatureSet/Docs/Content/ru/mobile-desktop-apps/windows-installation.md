# Руководство по установке на Windows

Установите OneUptime как настольное приложение на Windows для комплексного мониторинга и управления инцидентами.

## Способы установки

### Способ 1: Microsoft Edge (рекомендуется)

Edge обеспечивает наилучшую интеграцию PWA с нативными функциями Windows.

1. **Откройте OneUptime в Edge**

   - Запустите браузер Microsoft Edge
   - Перейдите по URL вашего экземпляра OneUptime
   - Войдите в свою учётную запись OneUptime
   - Дождитесь полной загрузки страницы

2. **Установите приложение**

   - Найдите **значок установки** (⊞) в адресной строке
   - Нажмите кнопку **«Установить OneUptime»**
   - Или нажмите **меню с тремя точками** → **Приложения** → **Установить этот сайт как приложение**

3. **Настройка установки**

   - **Имя приложения**: измените при желании (по умолчанию: OneUptime)
   - **Меню «Пуск»**: выберите, добавлять ли в меню «Пуск»
   - **Панель задач**: вариант закрепления на панели задач
   - **Рабочий стол**: создание ярлыка на рабочем столе

4. **Завершение установки**
   - Нажмите **«Установить»**
   - OneUptime откроется в собственном окне
   - Найдите его в меню «Пуск» в разделе установленных приложений

### Способ 2: Google Chrome

Chrome предлагает отличную поддержку PWA с широкой интеграцией в рабочий стол.

1. **Откройте OneUptime в Chrome**

   - Запустите Google Chrome
   - Перейдите на ваш экземпляр OneUptime
   - Убедитесь, что вы вошли в систему
   - Дождитесь полной загрузки страницы

2. **Установка через адресную строку**

   - Найдите **значок установки** (⊞) в адресной строке
   - Нажмите **«Установить OneUptime»**
   - Или воспользуйтесь меню: **три точки** → **Дополнительные инструменты** → **Создать ярлык**

3. **Параметры установки**

   - Установите флажок **«Открыть как окно»** для опыта приложения
   - При желании задайте имя приложения
   - Нажмите **«Установить»** или **«Создать»**

4. **Запуск приложения**
   - Найдите OneUptime в меню «Пуск» Windows
   - Или запустите через ярлык на рабочем столе
   - Приложение открывается в отдельном окне

### Способ 3: Firefox

Firefox поддерживает установку PWA с базовой интеграцией в рабочий стол.

1. **Откройте OneUptime в Firefox**

   - Запустите браузер Firefox
   - Перейдите по URL OneUptime
   - Завершите процесс входа

2. **Установите PWA**
   - Найдите **приглашение к установке** или баннер
   - Или нажмите **меню** → **Установить**
   - При наличии нажмите **«На экран «Домой»»** или аналогичный вариант

### Настройка запуска

1. **Автозапуск**: настройте автоматический запуск OneUptime вместе с Windows
   - Правая кнопка на панели задач → Диспетчер задач → Автозагрузка
   - Включите OneUptime при желании
2. **Размер по умолчанию**: установите предпочтительный размер и положение окна

### Настройка уведомлений

1. **Уведомления Windows**

   - Параметры → Система → Уведомления и действия
   - Найдите OneUptime и настройте параметры оповещений
   - Включите всплывающие уведомления об инцидентах

2. **Помощник в работе**
   - Настройте параметры режима «Не беспокоить»
   - Разрешите критические уведомления OneUptime
   - Установите уровни приоритета для разных типов оповещений

## Устранение неполадок

### Проблемы с установкой

**Кнопка установки не появляется:**

```
Решения:
1. Убедитесь, что используете Edge или Chrome (рекомендуемые браузеры)
2. Проверьте HTTPS-подключение к экземпляру OneUptime
3. Очистите кеш и файлы cookie браузера
4. Обновите браузер до последней версии
5. Проверьте, соответствует ли сервер требованиям PWA
6. Временно отключите расширения браузера
```

**Установка завершается ошибкой или приложение вылетает:**

```
Решения:
1. Запустите браузер от имени администратора
2. Проверьте настройки контроля учётных записей пользователей (UAC) Windows
3. Убедитесь в наличии достаточного места на диске (минимум 100 МБ)
4. Временно отключите антивирусное программное обеспечение
5. Полностью очистите данные браузера
6. Перезапустите Windows и повторите попытку
```

**Приложение не появляется в меню «Пуск»:**

```
Решения:
1. Найдите «OneUptime» через поиск Windows
2. Проверьте, не установлено ли приложение под другим именем
3. Найдите в разделе «Недавно добавленные» в меню «Пуск»
4. Переустановите и убедитесь, что установлен флажок «Добавить в меню «Пуск»»
5. При необходимости создайте ярлык вручную
```

### Проблемы с уведомлениями

**Уведомления Windows не работают:**

```
Решения:
1. Параметры Windows → Система → Уведомления и действия
2. Включите уведомления для OneUptime
3. Проверьте настройки помощника в работе
4. Убедитесь, что разрешения на уведомления установлены в OneUptime
5. Сначала протестируйте с простым уведомлением
```

## Удаление

### Полное удаление

1. **Метод через «Параметры Windows»**

   - Параметры → Приложения → Приложения и возможности
   - Найдите «OneUptime»
   - Нажмите и выберите «Удалить»

2. **Метод через браузер**

   - Откройте Edge/Chrome
   - Перейдите на edge://apps/ или chrome://apps/
   - Найдите OneUptime
   - Нажмите «Параметры» → «Удалить»

3. **Метод через меню «Пуск»**
   - Нажмите правой кнопкой на OneUptime в меню «Пуск»
   - Выберите «Удалить»
   - Подтвердите удаление

## Обновления и техническое обслуживание

### Автоматические обновления

- OneUptime PWA обновляется автоматически при наличии подключения к интернету
- Ручное вмешательство не требуется
- Обновления применяются немедленно после перезапуска
- Критические исправления развёртываются мгновенно
