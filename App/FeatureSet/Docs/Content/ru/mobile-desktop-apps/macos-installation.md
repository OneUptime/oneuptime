# Руководство по установке на macOS

Установите OneUptime как нативное настольное приложение на macOS для удобного мониторинга и управления инцидентами.

## Способы установки

### Способ 1: Safari (рекомендуется для macOS)

Safari обеспечивает отличную интеграцию PWA с нативными функциями macOS.

1. **Откройте OneUptime в Safari**

   - Запустите браузер Safari
   - Перейдите по URL вашего экземпляра OneUptime
   - Войдите в свою учётную запись OneUptime
   - Дождитесь полной загрузки страницы

2. **Установите PWA**

   - Нажмите **Файл** в строке меню
   - Выберите **«Добавить в Dock»** (macOS Sonoma+)
   - Или найдите **значок установки** в адресной строке
   - Альтернативно: **Файл** → **«На экран «Домой»»** (более ранние версии macOS)

3. **Настройка установки**

   - **Имя приложения**: измените при желании (по умолчанию: OneUptime)
   - **Dock**: выберите добавление в Dock
   - **Launchpad**: добавьте в Launchpad для быстрого доступа

4. **Запуск приложения**
   - Найдите OneUptime в Dock, Launchpad или папке «Программы»
   - Нажмите для запуска в отдельном окне
   - Приложение работает независимо от браузера Safari

### Способ 2: Google Chrome

Chrome предлагает надёжную поддержку PWA с превосходной интеграцией в рабочий стол.

1. **Откройте OneUptime в Chrome**

   - Запустите Google Chrome
   - Перейдите на ваш экземпляр OneUptime
   - Убедитесь, что вы вошли в систему
   - Дождитесь полной загрузки страницы

2. **Установка через меню**

   - Найдите **значок установки** (⊞) в адресной строке
   - Нажмите **«Установить OneUptime»**
   - Или используйте **меню Chrome** → **Дополнительные инструменты** → **Создать ярлык**

3. **Параметры установки**

   - Установите флажок **«Открыть как окно»** для опыта нативного приложения
   - При необходимости задайте имя приложения
   - Нажмите **«Установить»** или **«Создать»**

4. **Доступ к приложению**
   - Найдите OneUptime в папке «Программы»
   - Или воспользуйтесь поиском Spotlight
   - Закрепите в Dock для быстрого доступа

### Способ 3: Microsoft Edge

Edge обеспечивает надёжную поддержку PWA с хорошей интеграцией в macOS.

1. **Откройте OneUptime в Edge**

   - Запустите Microsoft Edge
   - Перейдите по URL OneUptime
   - Завершите процесс входа

2. **Установите приложение**
   - Нажмите **меню с тремя точками** → **Приложения** → **Установить этот сайт как приложение**
   - Или найдите приглашение к установке в адресной строке
   - При желании задайте имя приложения
   - Нажмите **«Установить»**

### Параметры настройки

### Dock и Launchpad

1. **Позиция в Dock**: перетащите OneUptime в нужное место в Dock
2. **Размер в Dock**: измените размер значка в настройках Dock
3. **Организация в Launchpad**: создайте папку для приложений мониторинга
4. **Значки уведомлений**: отображение количества инцидентов на значке в Dock

### Строка меню и уведомления

1. **Центр уведомлений**

   - Системные настройки → Уведомления → OneUptime
   - Настройте стили и доставку оповещений
   - Установите уровни приоритета для разных типов инцидентов

2. **Интеграция со строкой меню**
   - Нативная строка меню для PWA в Safari
   - Пользовательские пункты меню для частых действий
   - Сочетания клавиш для распространённых задач

## Устранение неполадок

### Проблемы с установкой

**«Добавить в Dock» недоступно в Safari:**

```
Решения:
1. Убедитесь в наличии macOS Sonoma (14.0) или более поздней версии
2. Обновите Safari до последней версии
3. Попробуйте альтернативу: Файл → На экран «Домой»
4. Очистите кеш Safari и повторите попытку
5. Используйте Chrome или Edge как альтернативу
```

**PWA не устанавливается или вылетает:**

```
Решения:
1. Проверьте совместимость версии macOS
2. Убедитесь в наличии достаточного места на диске (100 МБ+)
3. Обновите браузер до последней версии
4. Очистите кеш и файлы cookie браузера
5. Временно отключите расширения браузера
6. Перезапустите Mac и повторите установку
```

**Приложение не отображается в папке «Программы»:**

```
Решения:
1. Проверьте наличие значка OneUptime в Launchpad
2. Выполните поиск через Spotlight (⌘+Пробел)
3. Загляните в раздел управления PWA в браузере
4. Попробуйте переустановить через другой браузер
5. Проверьте, не установлено ли приложение под другим именем
```

### Проблемы с уведомлениями

**Уведомления macOS не работают:**

```
Решения:
1. Системные настройки → Уведомления → OneUptime
2. Включите «Разрешить уведомления»
3. Установите подходящий стиль оповещений (баннеры/предупреждения)
4. Проверьте настройки режима «Не беспокоить»
5. Проверьте настройки уведомлений OneUptime
6. Предоставьте разрешения на уведомления при появлении запроса
```

## Удаление

### Полное удаление

1. **Метод через папку «Программы»**

   - Откройте папку «Программы»
   - Найдите OneUptime
   - Перетащите в Корзину или нажмите правой кнопкой → «Переместить в корзину»

2. **Метод через Dock**

   - Нажмите правой кнопкой на OneUptime в Dock
   - Выберите «Параметры» → «Убрать из Dock»
   - Затем удалите из папки «Программы»

3. **Управление PWA через браузер**
   - **Chrome**: chrome://apps/ → Найдите OneUptime → Удалить
   - **Edge**: edge://apps/ → Найдите OneUptime → Удалить
   - **Safari**: специальной страницы управления нет

## Обновления и техническое обслуживание

### Автоматические обновления

- OneUptime PWA обновляется автоматически при наличии подключения к интернету
- Обновления из App Store не требуются
- Новые функции становятся доступны немедленно
- Критические обновления применяются мгновенно

### Проблемы с производительностью

**Низкая производительность или высокая нагрузка на CPU:**

```
Решения:
1. Проверьте использование ресурсов в «Мониторе активности»
2. Закройте ненужные приложения
3. Убедитесь в достаточном объёме ОЗУ (рекомендуется 8 ГБ+)
4. Обновите macOS и браузер
5. Очистите кеш браузера и данные приложения
6. Перезапустите приложение OneUptime
```

**Утечки памяти или вылеты:**

```
Решения:
1. Следите за использованием памяти в «Мониторе активности»
2. Регулярно перезапускайте приложение OneUptime
3. Обновите браузер до последней версии
4. Полностью очистите кеш браузера
5. Проверьте журналы ошибок в приложении «Консоль»
6. Сообщайте о проблемах с журналами аварийных завершений
```

### Проблемы с отображением и окнами

**Проблемы с размером или положением окна:**

```
Решения:
1. Вручную измените размер и переместите окно
2. Используйте меню «Окно» → «Развернуть» (для PWA в Safari)
3. Сбросьте состояние окна, выйдя и снова открыв приложение
4. Проверьте масштаб отображения в «Системных настройках»
5. Попробуйте другое пространство рабочего стола или полноэкранный режим
```

**Приложение не реагирует:**

```
Решения:
1. Принудительно завершите: ⌘+Option+Esc → Выберите OneUptime
2. Или нажмите правой кнопкой на значок в Dock → «Принудительное завершение»
3. Перезапустите приложение
4. Проверьте наличие обновлений macOS и браузера
5. Очистите кеш приложения и переустановите при необходимости
```

### Полная очистка

Удалите все связанные данные:

```bash
# Очистка данных PWA в Safari (общие данные сайтов)
rm -rf ~/Library/Safari/Databases
rm -rf ~/Library/Caches/com.apple.Safari

# Очистка данных PWA в Chrome
rm -rf ~/Library/Application\ Support/Google/Chrome/Default/Web\ Applications

# Очистка данных PWA в Edge
rm -rf ~/Library/Application\ Support/Microsoft\ Edge/Default/Web\ Applications
```

## Обновления и техническое обслуживание

### Автоматические обновления

- OneUptime PWA обновляется автоматически при наличии подключения к интернету
- Обновления из App Store не требуются
- Новые функции становятся доступны немедленно
- Критические обновления применяются мгновенно

### Процесс ручного обновления

Принудительно обновите приложение:

1. **PWA в Safari**: обновите в браузере Safari
2. **PWA в Chrome**: нажмите правой кнопкой на приложение → «Перезагрузить» или ⌘+R
3. **Полное обновление**: закройте приложение, заново откройте браузер, перейдите в OneUptime

### Расписание технического обслуживания

Регулярное обслуживание для оптимальной производительности:

**Еженедельно:**

- Перезапускайте приложение OneUptime
- Очищайте кеш браузера при возникновении проблем
- Проверяйте наличие обновлений macOS

**Ежемесячно:**

- Проверяйте использование хранилища и очищайте при необходимости
- Обновляйте браузеры, если автоматическое обновление не выполняется
- Проверяйте работоспособность настроек уведомлений

## Интеграция с функциями macOS

### Интеграция с приложением «Команды»

Создайте пользовательские команды для OneUptime:

1. Откройте приложение **Команды**
2. Создайте **Новую команду**
3. Добавьте действие **«Открыть приложение»**
4. Выберите **OneUptime**
5. Добавьте в Siri для голосовой активации

### Интеграция с Automator

Автоматизируйте задачи OneUptime:

1. Запустите **Automator**
2. Создайте **Приложение** или **Рабочий процесс**
3. Добавьте действие **«Запустить приложение»**
4. Выберите OneUptime PWA
5. Добавьте дополнительные шаги автоматизации

### Интеграция с Terminal

Управляйте OneUptime через Terminal:

```bash
# Создайте псевдоним для быстрого запуска OneUptime
echo 'alias oneuptime="open -a \"OneUptime\""' >> ~/.zshrc

# Функция проверки состояния OneUptime
oneuptime_status() {
    if pgrep -f "OneUptime" > /dev/null; then
        echo "OneUptime is running"
    else
        echo "OneUptime is not running"
    fi
}
```

## Безопасность и конфиденциальность

### Функции безопасности macOS

1. **Gatekeeper**: убедитесь, что установка PWA выполняется из доверенных источников
2. **Защита целостности системы**: защищает системные файлы
3. **FileVault**: шифрование диска для защиты данных
4. **Keychain**: безопасное хранение учётных данных

### Соображения конфиденциальности

1. **Службы геолокации**: настройте при необходимости для мониторинга
2. **Камера/микрофон**: предоставьте разрешения по необходимости
3. **Запись экрана**: может потребоваться для отдельных функций мониторинга
4. **Доступ к сети**: обеспечьте правильную конфигурацию брандмауэра

### Рекомендации по использованию

1. **Регулярные обновления**: поддерживайте macOS и браузеры в актуальном состоянии
2. **Надёжная аутентификация**: используйте Touch ID/Face ID там, где доступно
3. **Сетевая безопасность**: используйте VPN для удалённого доступа к мониторингу
4. **Резервное копирование**: регулярные резервные копии Time Machine включают данные PWA
5. **Проверка разрешений**: регулярно проверяйте предоставленные разрешения
