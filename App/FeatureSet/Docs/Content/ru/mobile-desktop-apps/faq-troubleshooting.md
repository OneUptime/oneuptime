# Часто задаваемые вопросы и устранение неполадок

Часто задаваемые вопросы и решения для мобильных и десктопных приложений OneUptime (PWA).

## Общие вопросы

### Что такое Progressive Web App (PWA)?

Progressive Web App — это веб-приложение, использующее современные веб-технологии для обеспечения опыта, подобного нативному приложению. PWA могут устанавливаться непосредственно из браузеров без магазинов приложений, работать в автономном режиме, отправлять push-уведомления и интегрироваться с операционной системой вашего устройства.

### Почему OneUptime не использует традиционные магазины приложений?

OneUptime использует технологию PWA, поскольку она предлагает несколько преимуществ:
- **Мгновенные обновления**: Не нужно ждать одобрения магазина приложений или ручных обновлений
- **Кросс-платформенность**: Единая кодовая база работает на всех устройствах
- **Нет ограничений на размер загрузки**: Полный функционал без ограничений по размеру
- **Прямая дистрибуция**: Устанавливайте прямо из вашего экземпляра OneUptime
- **Всегда актуально**: У пользователей всегда последняя версия
- **Безопасность**: Те же преимущества безопасности, что и у веб-приложений


### Сколько места использует OneUptime PWA?

- **Начальная установка**: 10-20 МБ
- **Рост кэша**: 50-100 МБ при регулярном использовании
- **Максимальный кэш**: Обычно ограничен 200 МБ браузерами
- **Автоочистка**: Браузеры автоматически управляют хранилищем

### Поддерживает ли OneUptime PWA push-уведомления?

Да, OneUptime PWA поддерживает богатые push-уведомления:
- **Алерты инцидентов**: Уведомления об инцидентах в реальном времени
- **Обновления статуса**: Алерты об изменении статуса монитора
- **Пользовательские триггеры**: Настройка правил уведомлений
- **Богатый контент**: Изображения, действия и подробная информация
- **Обновления значка**: Счётчик непрочитанных на значке приложения

## Вопросы об установке

### Почему кнопка "Install" не появляется?

Распространённые причины и решения:
1. **Совместимость браузера**: Используйте Chrome, Edge или Safari
2. **Требуется HTTPS**: Убедитесь, что экземпляр OneUptime использует HTTPS
3. **Требования PWA**: Сервер должен соответствовать требованиям манифеста PWA
4. **Проблемы с кэшем**: Очистите кэш браузера и перезагрузите страницу
5. **Уже установлено**: Приложение может быть уже установлено
6. **Время ожидания**: Некоторым браузерам нужно 30+ секунд на странице

### Могу ли я установить приложение на несколько устройств?

Да! Вы можете установить OneUptime PWA на:
- Неограниченное количество устройств на пользователя
- Несколько браузеров на одном устройстве
- Различные операционные системы
- Общие/семейные устройства (с отдельными аккаунтами)

### Как обновить установленное приложение?

OneUptime PWA обновляется автоматически:
- **Автоматические обновления**: Приложение обновляется при посещении в режиме онлайн
- **Фоновые обновления**: Обновления загружаются в фоне
- **Немедленная доступность**: Новые функции доступны мгновенно
- **Не требует действий пользователя**: В отличие от приложений из магазина, ручные обновления не нужны

### Могу ли я настроить имя приложения при установке?

Да, при установке вы можете:
- Изменить имя приложения (по умолчанию: "OneUptime")
- Добавить название организации
- Использовать пользовательское соглашение об именовании
- Изменить метку значка (зависит от платформы)

### Как удалить OneUptime PWA?

Удаление зависит от платформы:

**Android:**
- Долгое нажатие на значок → Uninstall
- Settings → Apps → OneUptime → Uninstall

**iOS:**
- Долгое нажатие на значок → Remove App → Delete App

**Windows:**
- Settings → Apps → OneUptime → Uninstall
- Правый клик на элемент меню "Пуск" → Uninstall

**macOS:**
- Перетащить из Applications в Trash
- Правый клик на значок в Dock → Remove

**Linux:**
- Удалить из лаунчера приложений
- Удалить файл .desktop


## Вопросы об уведомлениях

### Почему я не получаю уведомления?

Распространённые проблемы с уведомлениями и их решения:

**Проверьте разрешения:**
```
1. Разрешения уведомлений браузера включены
2. Разрешения уведомлений операционной системы
3. Настройки уведомлений OneUptime настроены
4. Режим "Не беспокоить" отключён
```

**Специфично для платформы:**
- **Android**: Проверьте настройки оптимизации батареи
- **iOS**: Проверьте настройки уведомлений в приложении Settings
- **Windows**: Проверьте настройки Focus Assist
- **macOS**: Проверьте разрешения центра уведомлений
- **Linux**: Проверьте состояние демона уведомлений

### Могу ли я настроить звуки уведомлений?

Параметры настройки уведомлений:
- **Системные звуки**: Используйте настройки звука уведомлений ОС
- **Настройки браузера**: Настройте в настройках уведомлений браузера
- **Настройки OneUptime**: Установите предпочтения уведомлений в панели управления
- **Уровни приоритета**: Настройте разные звуки для уровней серьёзности

### Как временно отключить уведомления?

Временное отключение уведомлений:
- **Режим "Не беспокоить"**: Включите системный режим DND
- **Настройки браузера**: Временно отключите уведомления сайта
- **Панель управления OneUptime**: Приостановите уведомления в настройках
- **Режимы фокусировки**: Используйте режимы фокусировки/концентрации ОС

## Вопросы безопасности

### Безопасно ли OneUptime PWA?

Функции и соображения безопасности:
- **Шифрование HTTPS**: Все данные передаются безопасно
- **Политика одного источника**: Применяются ограничения безопасности браузера
- **Изолированная среда**: Работает в изолированной среде безопасности браузера
- **Регулярные обновления**: Патчи безопасности применяются автоматически
- **Нет root-доступа**: Ограниченный доступ к системе по сравнению с нативными приложениями


*Примечание: Конфиденциальные данные шифруются и соответствуют стандартам безопасности браузера.*

### Могу ли я использовать OneUptime PWA в корпоративных сетях?

Соображения о корпоративной сети:
- **Правила брандмауэра**: Обеспечьте доступ HTTPS (порт 443)
- **Конфигурация прокси**: Настройте параметры прокси браузера
- **Доверие сертификатам**: Установите корпоративные сертификаты при необходимости
- **Доступ через VPN**: Используйте VPN для удалённого доступа
- **Политики безопасности**: Соблюдайте требования IT-безопасности

## Устранение неполадок

### Проблемы с установкой

**Проблема**: Кнопка установки не появляется
```
Решения:
1. Подождите 30+ секунд на странице OneUptime
2. Обновите страницу и подождите снова
3. Очистите кэш и куки браузера
4. Попробуйте другой браузер (рекомендуется Chrome/Edge)
5. Убедитесь в наличии HTTPS-соединения (проверьте значок замка)
6. Проверьте, не установлено ли приложение уже
```

**Проблема**: Установка не выполняется или аварийно завершается
```
Решения:
1. Убедитесь в достаточном свободном месте (100 МБ+)
2. Закройте другие вкладки браузера и приложения
3. Обновите браузер до последней версии
4. Временно отключите расширения браузера
5. Попробуйте установку в приватном/инкогнито режиме
6. Перезапустите браузер и повторите попытку
```

**Проблема**: Приложение установлено, но не появляется
```
Решения:
1. Проверьте все места для запуска приложений
2. Найдите "OneUptime" в поиске устройства
3. Загляните в раздел управления приложениями браузера
4. Подождите 1-2 минуты для обновления системы
5. Перезагрузите устройство и проверьте снова
```

**Проблема**: Приложение часто аварийно завершается
```
Решения:
1. Обновите браузер до последней версии
2. Очистите все данные браузера для OneUptime
3. Отключите расширения браузера
4. Проверьте свободное место для хранения
5. Перезагрузите операционную систему
6. Переустановите OneUptime PWA
```

**Проблема**: Push-уведомления не работают
```
Решения:
1. Проверьте разрешения уведомлений в браузере
2. Проверьте системные настройки уведомлений
3. Сначала проверьте простое уведомление
4. Очистите данные уведомлений и повторно предоставьте разрешения
5. Проверьте настройки режима "Не беспокоить"/фокусировки
6. Проверьте конфигурацию уведомлений OneUptime
```

**Проблема**: Приложение не синхронизирует последние данные
```
Решения:
1. Потяните вниз для обновления (мобильные)
2. Нажмите Ctrl+F5 (Windows/Linux) или Cmd+R (Mac)
3. Закройте и снова откройте приложение
4. Очистите кэш приложения и перезагрузите
5. Проверьте сетевое соединение
```

### Проблемы, специфичные для платформы

**Проблемы Android:**
```
Проблема: Приложение не появляется в панели приложений
Решение: Проверьте раздел "Recently added", найдите в панели приложений

Проблема: Задержка уведомлений
Решение: Отключите оптимизацию батареи для браузера

Проблема: Приложение аварийно завершается при запуске
Решение: Очистите данные приложения Chrome, перезагрузите устройство
```

**Проблемы iOS:**
```
Проблема: Невозможно добавить на главный экран
Решение: Используйте браузер Safari, убедитесь в iOS 11.3+

Проблема: Значок приложения пропал
Решение: Проверьте все страницы главного экрана и библиотеку приложений

Проблема: Face ID не работает
Решение: Включите Face ID для Safari в настройках
```

**Проблемы Windows:**
```
Проблема: Приложение не появляется в меню Пуск
Решение: Найдите приложение через поиск, проверьте список установленных приложений

Проблема: Уведомления не отображаются
Решение: Проверьте настройки уведомлений Windows, включите для браузера

Проблема: Проблемы с размером окна
Решение: Измените размер вручную, приложение запомнит размеры
```

**Проблемы macOS:**
```
Проблема: Невозможно установить через Safari
Решение: Обновитесь до macOS Sonoma+, используйте File → Add to Dock

Проблема: Приложение не в папке Applications
Решение: Проверьте Launchpad, используйте поиск Spotlight

Проблема: Уведомления не работают
Решение: Проверьте System Preferences → Notifications
```

**Проблемы Linux:**
```
Проблема: Опция установки PWA отсутствует
Решение: Используйте Chrome/Chromium, убедитесь в поддержке desktop environment

Проблема: Значок не появляется в лаунчере
Решение: Обновите базу данных рабочего стола, проверьте файл .desktop

Проблема: Звуковые уведомления не работают
Решение: Проверьте PulseAudio, убедитесь в наличии аудиоразрешений браузера
```

### Сообщения об ошибках

**"This site cannot be installed"**
```
Причины:
- Экземпляр OneUptime не соответствует требованиям PWA
- Отсутствует или недействителен манифест веб-приложения
- HTTPS не настроен должным образом
- Браузер не поддерживает установку PWA

Решения:
- Свяжитесь с администратором для проверки настройки PWA
- Попробуйте другой браузер
- Проверьте консоль браузера для получения подробных ошибок
```
