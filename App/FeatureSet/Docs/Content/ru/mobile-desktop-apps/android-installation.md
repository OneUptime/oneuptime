# Руководство по установке на Android

Установите OneUptime как нативное приложение на устройство Android для наилучшего опыта мониторинга.

## Методы установки

### Метод 1: Браузер Chrome (рекомендуется)

1. **Откройте OneUptime в Chrome**
   - Запустите Google Chrome на вашем устройстве Android
   - Перейдите по URL вашего экземпляра OneUptime
   - Дождитесь полной загрузки страницы

2. **Запрос на установку**
   - Ищите баннер "Add to Home Screen" внизу
   - Нажмите "Install" или "Add to Home Screen"
   - Если запрос не появляется, нажмите меню из трёх точек (⋮) в верхнем правом углу

3. **Ручная установка через меню**
   - Нажмите меню Chrome (три точки)
   - Выберите "Add to Home Screen" или "Install App"
   - При желании настройте имя приложения
   - Нажмите "Add" для подтверждения

4. **Запуск приложения**
   - Найдите значок OneUptime на главном экране или в панели приложений
   - Нажмите для запуска приложения в полноэкранном режиме

### Метод 2: Samsung Internet

1. **Откройте OneUptime**
   - Запустите браузер Samsung Internet
   - Перейдите на ваш экземпляр OneUptime
   - Дождитесь полной загрузки страницы

2. **Добавить на главный экран**
   - Нажмите кнопку меню (три линии)
   - Выберите "Add page to" → "Home screen"
   - Введите имя приложения и нажмите "Add"

3. **Запуск**
   - Найдите значок приложения на главном экране
   - Нажмите, чтобы открыть OneUptime в режиме приложения

### Метод 3: Firefox

1. **Откройте OneUptime**
   - Запустите браузер Firefox
   - Перейдите по URL OneUptime
   - Дождитесь полной загрузки страницы

2. **Установка**
   - Нажмите меню из трёх точек
   - Выберите "Install" (если доступно)
   - Или выберите "Add to Home Screen"
   - Подтвердите установку

### Параметры настройки

### Имя приложения
- При установке вы можете настроить имя приложения
- По умолчанию: "OneUptime"
- Рекомендуется: Оставить как "OneUptime" или добавить название компании

### Настройки уведомлений
1. **Предоставление разрешений**
   - Разрешите уведомления при появлении запроса
   - Перейдите в Settings → Apps → OneUptime → Notifications
   - Включите все категории уведомлений для наилучшего опыта

2. **Настройка алертов**
   - Настройте, какие инциденты вызывают уведомления
   - Установите уровни приоритета уведомлений
   - Выберите настройки звука и вибрации

## Устранение неполадок

### Проблемы с установкой

**"Add to Home Screen" не появляется:**
```
1. Очистите кэш и куки браузера
2. Убедитесь, что используется HTTPS (защищённое соединение)
3. Подождите 2-3 минуты на странице перед поиском запроса
4. Проверьте, выполнены ли требования PWA в вашем экземпляре OneUptime
```

**Установка не выполняется:**
```
1. Освободите место для хранения (необходимо минимум 50 МБ)
2. Обновите браузер до последней версии
3. Перезапустите браузер и повторите попытку
4. Попробуйте другой браузер (рекомендуется Chrome)
```

**Значок приложения не появляется:**
```
1. Проверьте главный экран и панель приложений
2. Поищите в разделе "Recently added"
3. Найдите "OneUptime" в панели приложений
4. При необходимости переустановите
```

### Проблемы с уведомлениями

**Уведомления не поступают:**
```
1. Проверьте разрешения уведомлений:
   - Settings → Apps → OneUptime → Permissions → Notifications
2. Убедитесь, что уведомления включены в панели управления OneUptime
3. Проверьте настройки режима "Не беспокоить"
4. Убедитесь, что оптимизация батареи не блокирует OneUptime
```

**Уведомления задерживаются:**
```
1. Отключите оптимизацию батареи для OneUptime:
   - Settings → Apps → OneUptime → Battery → Optimize battery usage
2. Разрешите фоновую активность
3. Проверьте настройки экономии трафика
```

## Удаление

### Удаление приложения
1. **Удерживайте** значок OneUptime на главном экране
2. Выберите **"Uninstall"** или перетащите в корзину
3. Подтвердите удаление

### Альтернативный метод
1. Перейдите в **Settings → Apps**
2. Найдите **"OneUptime"**
3. Нажмите **"Uninstall"**
4. Подтвердите удаление

## Обновления и обслуживание

### Автоматические обновления
OneUptime PWA обновляется автоматически:
- **Автоматические обновления**: Приложение обновляется при посещении в режиме онлайн
- **Ручные обновления не требуются**: В отличие от приложений из магазина, никаких действий пользователя не требуется
- **Мгновенные обновления**: Новые функции доступны немедленно
- **Безопасный откат**: Плохие обновления можно быстро откатить

## Расширенная конфигурация

### Параметры разработчика
Для опытных пользователей, желающих проверить PWA:
1. Включите режим разработчика в Android
2. Подключитесь к компьютеру через ADB
3. Используйте Chrome DevTools для удалённой отладки

### Конфигурация сети
- Настройте VPN, если получаете доступ к внутреннему экземпляру OneUptime
- Настройте параметры прокси, если это требует ваша организация
- Убедитесь, что брандмауэр разрешает ресурсы PWA

## Рекомендации по безопасности

1. **Только HTTPS**: Устанавливайте приложение только с защищённых экземпляров OneUptime
2. **Официальные URL**: Проверяйте, что устанавливаете приложение с официального URL OneUptime вашей организации
3. **Разрешения**: Предоставляйте только необходимые разрешения
4. **Обновления**: Поддерживайте актуальность вашей ОС Android и браузеров
