# Мобильные и десктопные приложения OneUptime

OneUptime предлагает нативные мобильные и десктопные приложения через технологию Progressive Web App (PWA). Эти приложения обеспечивают seamless-опыт на всех основных платформах без необходимости загрузки из магазинов приложений.

## Что такое Progressive Web App (PWA)?

Progressive Web App — это веб-приложение, использующее современные возможности Интернета для предоставления пользователям опыта, подобного нативному приложению.

## Начало работы

1. Откройте ваш экземпляр OneUptime в поддерживаемом браузере
2. Ищите запрос "Install" или "Add to Home Screen"
3. Следуйте инструкциям по установке, специфичным для вашей платформы
4. Запустите приложение из панели приложений или рабочего стола вашего устройства

## Устранение неполадок

### Распространённые проблемы

**Опция установки не появляется:**
- Убедитесь, что используете поддерживаемый браузер
- Проверьте, что ваш экземпляр OneUptime обслуживается через HTTPS
- Попробуйте обновить страницу или очистить кэш браузера

**Push-уведомления не работают:**
- Предоставьте разрешения уведомлений при появлении запроса
- Проверьте настройки уведомлений браузера
- Убедитесь, что экземпляр OneUptime настроен с уведомлениями


## Поддержка

Если вы столкнулись с проблемами с OneUptime PWA:

1. Ознакомьтесь с разделом устранения неполадок выше
2. Убедитесь, что ваш браузер соответствует минимальным требованиям
3. Свяжитесь со службой поддержки через панель управления OneUptime
4. Сообщите о проблемах в нашем [репозитории GitHub](https://github.com/OneUptime/oneuptime)
