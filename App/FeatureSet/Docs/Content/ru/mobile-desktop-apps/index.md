# Мобильные и настольные приложения OneUptime

OneUptime предоставляет два способа использования платформы за пределами браузера:

- **Нативные мобильные приложения** для iOS и Android, опубликованные в **Apple App Store** и **Google Play**. Они доставляют дежурные уведомления, оповещения об инцидентах и действия подтверждения непосредственно на ваш телефон.
- **Устанавливаемые настольные приложения** для Windows, macOS и Linux, предоставляемые в виде Progressive Web App (PWA), которое устанавливается непосредственно из вашего браузера. Они обеспечивают панели управления OneUptime собственное окно, значок и поверхность уведомлений на вашем компьютере.

## Мобильные (нативные приложения)

Приложение **OneUptime On-Call** — это нативное приложение, созданное на React Native. Оно распространяется через официальные магазины, поэтому вы получаете автоматические обновления, push-уведомления и биометрическую разблокировку.

- **iOS** — [Загрузить в App Store](https://apps.apple.com/us/app/oneuptime-on-call/id6759615391). Требуется iOS 15.0 или более поздняя версия. См. [Руководство по установке для iOS](./ios-installation.md).
- **Android** — [Получить в Google Play](https://play.google.com/store/apps/details?id=com.oneuptime.oncall). Требуется Android 8.0 или более поздняя версия. Также доступна прямая [загрузка APK](https://github.com/OneUptime/oneuptime/releases/latest/download/oneuptime-on-call-android-app.apk) для устройств без Google Play. См. [Руководство по установке для Android](./android-installation.md).

## Настольные (Progressive Web App)

Веб-панель управления OneUptime является Progressive Web App, поэтому вы можете установить её как настольное приложение из современного браузера без обращения к какому-либо магазину.

- [Установка для Windows](./windows-installation.md)
- [Установка для macOS](./macos-installation.md)
- [Установка для Linux](./linux-installation.md)

### Начало работы с настольным приложением

1. Откройте ваш экземпляр OneUptime в браузере на базе Chromium (Chrome, Edge) или Safari.
2. Найдите кнопку **Установить** в адресной строке или в меню **Файл → Добавить в Dock / Приложения → Установить этот сайт как приложение**.
3. Запустите установленное приложение из меню «Пуск», Launchpad или панели запуска приложений.

### Устранение неполадок настольного приложения

**Не отображается опция установки:**
- Убедитесь, что вы используете поддерживаемый браузер.
- Убедитесь, что ваш экземпляр OneUptime обслуживается по HTTPS.
- Обновите страницу или очистите кэш браузера.

**Push-уведомления не работают:**
- Предоставьте разрешения на уведомления, когда браузер запросит их.
- Проверьте настройки уведомлений вашей операционной системы для браузера.
- Пользователи self-hosted установок: убедитесь, что push-уведомления настроены в вашем экземпляре OneUptime.
