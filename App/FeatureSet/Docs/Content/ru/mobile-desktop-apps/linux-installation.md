# Руководство по установке на Linux

Установите OneUptime как настольное приложение на дистрибутивы Linux для комплексного мониторинга и управления инцидентами.

## Способы установки

### Способ 1: Google Chrome/Chromium (рекомендуется)

Chrome и Chromium обеспечивают лучший опыт PWA на Linux с нативной интеграцией в рабочий стол.

#### Шаги по установке PWA:

1. **Откройте OneUptime в Chrome/Chromium**

   - Запустите браузер
   - Перейдите по URL вашего экземпляра OneUptime
   - Войдите в свою учётную запись OneUptime
   - Дождитесь полной загрузки страницы

2. **Установите PWA**

   - Найдите **значок установки** (⊞) в адресной строке
   - Нажмите **«Установить OneUptime»**
   - Или воспользуйтесь **меню Chrome** (⋮) → **Дополнительные инструменты** → **Создать ярлык**

3. **Параметры установки**

   - Установите флажок **«Открыть как окно»** для опыта нативного приложения
   - При желании задайте имя приложения
   - Выберите создание ярлыка на рабочем столе
   - Нажмите **«Установить»** или **«Создать»**

4. **Запуск приложения**
   - Найдите OneUptime в запускателе приложений
   - Или воспользуйтесь ярлыком на рабочем столе
   - Приложение открывается в отдельном окне

### Способ 2: Firefox

Firefox поддерживает установку PWA на Linux с базовой интеграцией в рабочий стол.

1. **Установка PWA**:
   - Откройте OneUptime в Firefox
   - Найдите баннер или приглашение к установке
   - Нажмите **«Установить»** при появлении
   - Примечание: интеграция с рабочим столом ограничена по сравнению с Chrome

### Способ 3: Microsoft Edge

Edge доступен для Linux и обеспечивает хорошую поддержку PWA.

1. **Установка PWA**: следуйте тем же шагам, что и для Chrome

## Обновления и техническое обслуживание

### Автоматические обновления

OneUptime PWA обновляется автоматически:

- Обновления применяются при обновлении приложения браузером
- Критические обновления безопасности развёртываются немедленно
- Ручное вмешательство не требуется

## Удаление

### Удаление через браузер

```bash
# Управление PWA в Chrome
google-chrome chrome://apps/

# Удаление всех связанных данных браузера OneUptime
rm -rf ~/.config/google-chrome/Default/Local\ Storage/leveldb/
rm -rf ~/.cache/google-chrome/Default/
```
