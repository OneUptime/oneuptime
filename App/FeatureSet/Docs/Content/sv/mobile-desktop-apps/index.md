# OneUptime Mobil- och Skrivbordsappar

OneUptime erbjuder inbyggda mobil- och skrivbordsapplikationer via Progressive Web App (PWA)-teknik. Dessa appar ger en sömlös upplevelse på alla större plattformar utan att kräva nedladdningar från appbutiker.

## Vad är en Progressive Web App (PWA)?

En Progressive Web App är en webbapplikation som använder moderna webbfunktioner för att leverera en app-liknande upplevelse till användarna.

## Kom igång

1. Besök din OneUptime-instans i en webbläsare som stöds
2. Leta efter prompten "Installera" eller "Lägg till på startskärmen"
3. Följ de plattformsspecifika installationsstegen
4. Starta appen från enhetens applåda eller skrivbord

## Felsökning

### Vanliga problem

**Installationsalternativet visas inte:**
- Se till att du använder en webbläsare som stöds
- Kontrollera att din OneUptime-instans serveras via HTTPS
- Prova att uppdatera sidan eller rensa webbläsarcachen

**Push-aviseringar fungerar inte:**
- Bevilja aviserings behörigheter när du uppmanas
- Kontrollera webbläsarens aviseringsinställningar
- Se till att OneUptime-instansen har aviseringar konfigurerade


## Support

Om du stöter på problem med OneUptime PWA:

1. Kontrollera felsökningsavsnittet ovan
2. Verifiera att din webbläsare uppfyller minimikraven
3. Kontakta supporten via din OneUptime-instrumentpanel
4. Rapportera problem i vårt [GitHub-repositorie](https://github.com/OneUptime/oneuptime)
