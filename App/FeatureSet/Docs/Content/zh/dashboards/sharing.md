# 共享与公开仪表板

大多数仪表板对你的项目是私有的——只有已登录的项目成员能看到。但 OneUptime 也允许你把仪表板发布到公开 URL 上，可选地用密码保护、用 IP 限制、并托管在自定义域名上。本页覆盖这四方面。

## 私有仪表板（默认）

默认情况下，只有已登录且是项目成员的用户才能访问仪表板。URL 形如 `https://oneuptime.com/dashboards/<id>/view`。直接访问需要身份认证以及该仪表板上相应的读取权限。

在项目内，所有权和标签决定谁能看到什么——见 [仪表板配置与权限](/docs/dashboards/configuration)。

## 公开仪表板

在 **仪表板 → 设置** 下，打开 **Public Dashboard**。仪表板现在有了第二个无需登录的 URL。把它分享给供应商、合作伙伴、客户，或者贴在公开 README 里。

公开仪表板：

- 只在 **View** 模式下渲染。公开访客无法编辑、无法（除时间范围 URL 外）改动、也看不到小部件面板。
- 包含你定义的变量——访客可以像内部用户一样从下拉中选择。
- 带上你在设置中配置的 **品牌**：页面标题、页面描述、Logo 文件、favicon。它们会出现在浏览器标签和社交预览里。

把启用 **Public Dashboard** 当作发布一个网页对待。仪表板上的每个小部件现在都对全世界可读。翻开关之前先审一遍画布上有什么。

## 主密码

要用密码而不是完全开放来给公开仪表板加门槛：

1. 启用 **Public Dashboard**。
2. 启用 **Master Password**。
3. 设置密码。

访客在仪表板渲染前会看到密码提示。密码在静态存储时是哈希过的；只保存哈希。

什么时候用主密码：

- 你想跟合作伙伴或客户分享，但又不希望 URL 一旦泄露就有效。
- 仪表板是"半公开"的——开放到不必给每个查看者建 OneUptime 账号，但又不至于放到开放互联网上。

更高价值的门槛（按查看者账号、谁看过什么的审计）应该让仪表板保持私有，并以只读成员身份邀请查看者加入项目。

## IP 允许名单

在 **Scale** 计划上，你可以把公开仪表板限制到一组源 IP 或 CIDR 范围。在 **仪表板 → 设置 → IP Whitelist** 下配置列表。

什么时候用 IP 允许名单：

- 仪表板应该只能从你的办公室或 VPN 访问。
- 供应商门户应该只能从他们公布的出口 IP 访问。
- 你想在主密码之上加一层纵深防御。

来自其它 IP 的请求会收到 403。

## 自定义域名

开箱即用时，公开仪表板托管在 `oneuptime.com` 上。要在你自己的子域上托管（例如 `dashboard.acme.com`）：

1. 在你的 DNS 上添加一条 CNAME 记录，把子域指向 OneUptime 公布的目标。
2. 在 **仪表板 → 设置 → Custom Domains** 下添加该域名。
3. 验证 DNS 记录（OneUptime 会帮你检查）。
4. 验证通过后，仪表板同时可在 OneUptime URL 和你的自定义域名上访问。

自定义域名适用于：

- 走你自己品牌的面向客户的仪表板。
- 联合品牌的合作伙伴仪表板。
- 公开健康页面的 SEO。

如果你向多个受众提供同一份内容，可以把多个自定义域名绑到一个仪表板上。

## 公开仪表板的品牌

在 **仪表板 → 设置** 下配置：

- **Page title** — `<title>` 标签和访客看到的标题。
- **Page description** — 被搜索引擎和社交预览使用的元描述。
- **Logo file** — 上传 PNG/SVG；显示在仪表板头部。
- **Favicon** — 上传；显示在浏览器标签。

品牌只对公开模式渲染生效。内部查看者始终看到 OneUptime 的品牌。

## 嵌入

你可以在自己的网站上用 `<iframe>` 嵌入公开仪表板：

```html
<iframe src="https://dashboard.acme.com/view"
        width="100%" height="800"
        frameborder="0"></iframe>
```

如果你嵌入了主密码保护的仪表板，访客仍然会在 iframe 中看到密码提示。

## 带变量状态的可分享 URL

仪表板 URL 把当前的变量选择和时间范围编码为查询参数。调整下拉、复制 URL、粘到聊天里——接收者看到的是相同视图，包括你当时看的时间范围。

这是把同事指向"事件开始时的那个仪表板"最快的方式——钉时间范围、复制、粘贴。
