# iOS 安装指南

在您的 iPhone 或 iPad 上将 OneUptime 安装为原生应用，随时随地进行无缝监控。

## 安装方法

### 方法一：Safari（推荐）

Safari 在 iOS 设备上提供最佳的 PWA 体验。

1. **在 Safari 中打开 OneUptime**
   - 在 iOS 设备上启动 Safari
   - 导航至您的 OneUptime 实例 URL
   - 等待页面完全加载
   - 确保您已登录 OneUptime 账号

2. **访问共享菜单**
   - 点击底部工具栏中的 **共享按钮**（带向上箭头的方形图标）
   - 在共享选项中滚动找到"添加到主屏幕"

3. **添加到主屏幕**
   - 点击 **"添加到主屏幕"**
   - 自定义应用名称（默认：「OneUptime」）
   - 点击右上角的 **"添加"**

4. **启动应用**
   - 在主屏幕上找到 OneUptime 图标
   - 点击以全屏应用模式启动

### 方法二：Chrome 浏览器

虽然 Chrome 可以使用，但 Safari 是 iOS PWA 体验的最佳选择。

1. **在 Chrome 中打开 OneUptime**
   - 启动 Chrome 浏览器
   - 前往您的 OneUptime 实例
   - 允许页面完全加载

2. **添加到主屏幕**
   - 点击 **三点菜单**（更多选项）
   - 选择 **"添加到主屏幕"**
   - 根据需要自定义应用名称
   - 点击 **"添加"**

### 方法三：其他浏览器

Firefox、Edge 和其他浏览器支持基本的 PWA 安装：

1. **打开 OneUptime**
   - 启动您的首选浏览器
   - 导航至 OneUptime URL
   - 等待页面完全加载

2. **查找安装选项**
   - 在浏览器菜单中查找"添加到主屏幕"或"安装"
   - 按照特定浏览器的安装提示操作

### 自定义选项

### 应用图标和名称
- **自定义名称**：安装时或之后更改
- **图标位置**：整理到文件夹或特定主屏幕页面
- **角标通知**：显示未读事件数量

### 通知配置
1. **启用通知**
   - 出现提示时，点击 **"允许"** 以接收通知
   - 或前往 设置 → 通知 → OneUptime
   - 启用所有通知类型以实现全面监控

2. **自定义告警样式**
   - **锁屏**：在锁定设备上显示事件告警
   - **横幅样式**：选择临时或持续横幅
   - **声音**：自定义通知声音和振动
   - **紧急告警**：为高优先级事件启用（需要权限）

## 故障排查

### 安装问题

**看不到"添加到主屏幕"：**
```
解决方案：
1. 确保使用 Safari（兼容性最佳）
2. 刷新页面并等待 30 秒
3. 检查您是否在正确的 OneUptime URL 上
4. 验证 HTTPS 连接（查看锁形图标）
5. 清除 Safari 缓存：设置 → Safari → 清除历史记录和网站数据
```

**安装完成但未出现图标：**
```
解决方案：
1. 检查所有主屏幕页面
2. 在应用程序库中查找（向左滑动到最后一个主屏幕页面）
3. 使用 Spotlight 搜索查找「OneUptime」
4. 重启设备后再检查
5. 如有必要，重新安装
```

**启动时应用崩溃：**
```
解决方案：
1. 强制关闭并重新打开应用
2. 重启 iOS 设备
3. 清除 Safari 缓存并重新安装
4. 确保 iOS 版本为 11.3 或更高
5. 释放设备存储空间
```

### 通知问题

**未收到推送通知：**
```
检查以下设置：
1. 设置 → 通知 → OneUptime → 允许通知
2. 设置 → 屏幕时间 → 内容和隐私限制 → 允许的应用
3. 勿扰模式设置
4. 检查 OneUptime 控制台中的通知设置
5. 退出并重新登录 OneUptime
```

**通知延迟或丢失：**
```
解决方案：
1. 保持应用在后台运行（不要强制关闭）
2. 在关键监控期间禁用低电量模式
3. 检查后台应用刷新：设置 → 通用 → 后台应用刷新
4. 确保有足够的存储空间
```

## 卸载

### 从主屏幕移除
1. **长按** OneUptime 应用图标
2. 点击 **"移除应用"**
3. 选择 **"删除应用"**
4. 确认删除

2. 在应用列表中找到 **OneUptime**
3. 点击 **"删除应用"**
4. 确认移除

## 高级配置

### 快捷指令应用集成
为 OneUptime 创建自定义 Siri 快捷指令：
1. 打开 **快捷指令** 应用
2. 创建 **新快捷指令**
3. 添加 **"打开应用"** 操作
4. 选择 **OneUptime**
5. 添加语音短语，如"检查系统状态"

### 专注模式
将 OneUptime 与 iOS 专注模式集成：
1. **设置 → 专注**
2. 选择或创建专注模式
3. **应用 → 添加应用 → OneUptime**
4. 为不同专注状态配置通知行为

### 屏幕时间管理
配置适当的屏幕时间设置：
1. **设置 → 屏幕时间 → 应用限额**
2. 将 OneUptime 添加到"效率"类别
3. 根据需要设置适当的使用时间限制
