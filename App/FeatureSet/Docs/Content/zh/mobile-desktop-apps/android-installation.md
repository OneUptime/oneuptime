# Android 安装指南

在您的 Android 设备上将 OneUptime 安装为原生应用，以获得最佳监控体验。

## 安装方法

### 方法一：Chrome 浏览器（推荐）

1. **在 Chrome 中打开 OneUptime**
   - 在您的 Android 设备上启动 Google Chrome
   - 导航至您的 OneUptime 实例 URL
   - 等待页面完全加载

2. **安装提示**
   - 查找底部的"添加到主屏幕"横幅
   - 点击"安装"或"添加到主屏幕"
   - 如果没有看到提示，点击右上角的三点菜单（⋮）

3. **通过菜单手动安装**
   - 点击 Chrome 菜单（三个点）
   - 选择"添加到主屏幕"或"安装应用"
   - 根据需要自定义应用名称
   - 点击"添加"确认

4. **启动应用**
   - 在主屏幕或应用抽屉中找到 OneUptime 图标
   - 点击以全屏模式启动应用

### 方法二：Samsung Internet

1. **打开 OneUptime**
   - 启动 Samsung Internet 浏览器
   - 前往您的 OneUptime 实例
   - 等待页面完全加载

2. **添加到主屏幕**
   - 点击菜单按钮（三条横线）
   - 选择"将页面添加到" → "主屏幕"
   - 输入应用名称并点击"添加"

3. **启动**
   - 在主屏幕上找到应用图标
   - 点击以应用模式打开 OneUptime

### 方法三：Firefox

1. **打开 OneUptime**
   - 启动 Firefox 浏览器
   - 导航至您的 OneUptime URL
   - 允许页面完全加载

2. **安装**
   - 点击三点菜单
   - 选择"安装"（如果可用）
   - 或选择"添加到主屏幕"
   - 确认安装

### 自定义选项

### 应用名称
- 安装过程中，您可以自定义应用名称
- 默认：「OneUptime」
- 建议：保持「OneUptime」或添加您的公司名称

### 通知设置
1. **授予权限**
   - 出现提示时允许通知
   - 前往 设置 → 应用 → OneUptime → 通知
   - 启用所有通知类别以获得最佳体验

2. **自定义告警**
   - 配置哪些事件触发通知
   - 设置通知优先级
   - 选择声音和振动偏好

## 故障排查

### 安装问题

**未出现"添加到主屏幕"提示：**
```
1. 清除浏览器缓存和 Cookie
2. 确保使用 HTTPS（安全连接）
3. 在页面上等待 2-3 分钟后再查找提示
4. 检查您的 OneUptime 实例是否满足 PWA 要求
```

**安装失败：**
```
1. 释放存储空间（至少需要 50MB）
2. 将浏览器更新到最新版本
3. 重启浏览器后重试
4. 尝试其他浏览器（推荐使用 Chrome）
```

**应用图标未出现：**
```
1. 检查主屏幕和应用抽屉
2. 在"最近添加"的应用部分查找
3. 在应用抽屉中搜索「OneUptime」
4. 如有必要，重新安装
```

### 通知问题

**未收到通知：**
```
1. 检查通知权限：
   - 设置 → 应用 → OneUptime → 权限 → 通知
2. 确保在 OneUptime 控制台中启用了通知
3. 检查勿扰模式设置
4. 验证电池优化设置是否阻止了 OneUptime
```

**通知延迟：**
```
1. 为 OneUptime 禁用电池优化：
   - 设置 → 应用 → OneUptime → 电池 → 优化电池使用量
2. 允许后台活动
3. 检查省流量设置
```

## 卸载

### 删除应用
1. **长按**主屏幕上的 OneUptime 图标
2. 选择 **"卸载"** 或拖到垃圾桶
3. 确认删除

### 替代方法
1. 前往 **设置 → 应用**
2. 找到 **「OneUptime」**
3. 点击 **"卸载"**
4. 确认删除

- **无需手动更新**：与应用商店应用不同，无需用户操作
- **即时更新**：新功能立即可用

## 高级配置

### 开发者选项
对于想要检查 PWA 的高级用户：
1. 在 Android 中启用开发者选项
2. 使用 ADB 连接到电脑
3. 使用 Chrome DevTools 进行远程调试

### 网络配置
- 如果访问内部 OneUptime 实例，请配置 VPN
- 如果您的组织需要，请设置代理设置
- 确保防火墙允许 PWA 资源

## 最佳实践

### 获得最佳性能
1. **首次启动**：首次设置时需要在线
2. **定期使用**：定期打开应用以保持缓存新鲜
3. **存储管理**：保持足够的可用存储空间
4. **网络**：初次安装和重大更新时使用 Wi-Fi

### 安全建议
1. **仅限 HTTPS**：仅从安全的 OneUptime 实例安装
2. **官方 URL**：验证您是从组织的官方 OneUptime URL 安装
3. **权限**：仅授予必要权限
4. **更新**：保持 Android 操作系统和浏览器为最新版本
