# OneUptime 移动端和桌面端应用

OneUptime 通过渐进式 Web 应用（PWA）技术提供类原生的移动端和桌面端应用程序。这些应用在所有主流平台上提供无缝体验，无需从应用商店下载。

## 什么是渐进式 Web 应用（PWA）？

渐进式 Web 应用是一种使用现代 Web 能力向用户提供类应用程序体验的 Web 应用程序。

## 入门指南

1. 在支持的浏览器中访问您的 OneUptime 实例
2. 查找"安装"或"添加到主屏幕"提示
3. 按照特定平台的安装步骤操作
4. 从设备的应用抽屉或桌面启动应用

## 故障排查

### 常见问题

**安装选项未出现：**
- 确保使用的是支持的浏览器
- 检查您的 OneUptime 实例是否通过 HTTPS 提供服务
- 尝试刷新页面或清除浏览器缓存

**推送通知不工作：**
- 出现提示时授予通知权限
- 检查浏览器通知设置
- 确保 OneUptime 实例已配置通知


## 支持

如果您在使用 OneUptime PWA 时遇到任何问题：

1. 查看上方的故障排查部分
2. 验证您的浏览器是否满足最低要求
3. 通过您的 OneUptime 控制台联系支持
4. 在我们的 [GitHub 仓库](https://github.com/OneUptime/oneuptime) 上报告问题
