# OneUptime Mobile और Desktop Apps

OneUptime, Progressive Web App (PWA) technology के माध्यम से native-like mobile और desktop applications प्रदान करता है। ये apps app stores से downloads की आवश्यकता के बिना सभी प्रमुख platforms पर seamless experience प्रदान करते हैं।

## Progressive Web App (PWA) क्या है?

एक Progressive Web App एक web application है जो users को app-like experience देने के लिए modern web capabilities का उपयोग करती है।

## शुरुआत करना

1. एक समर्थित browser में अपने OneUptime instance पर जाएं
2. "Install" या "Add to Home Screen" prompt देखें
3. platform-specific installation steps का पालन करें
4. अपने device के app drawer या desktop से app launch करें

## समस्या निवारण

### सामान्य समस्याएं

**Installation option नहीं दिख रहा:**
- सुनिश्चित करें कि आप एक समर्थित browser उपयोग कर रहे हैं
- जांचें कि आपका OneUptime instance HTTPS पर serve किया जा रहा है
- page refresh करने या browser cache clear करने की कोशिश करें

**Push notifications काम नहीं कर रहीं:**
- prompt होने पर notification permissions grant करें
- browser notification settings जांचें
- सुनिश्चित करें कि OneUptime instance में notifications configured हैं
