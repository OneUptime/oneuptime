# Helm Chart for OneUptime

This repository is used to host all the helm-charts for OneUptime.

### Helm Packages 

- *oneuptime* : Chart for deploying OneUptime app. [Read Docs here](oneuptime/README.md)
- *kubernetes-agent* : Chart for installing the OneUptime Kubernetes monitoring agent into any cluster (standard, GKE Autopilot, EKS Fargate). [Read Docs here](kubernetes-agent/README.md)
