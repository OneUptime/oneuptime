OneUptime Helm Charts

The official Helm chart repository for OneUptime — deploy the full observability platform and the Kubernetes monitoring agent to your cluster.

Website GitHub Chart Docs Artifact Hub

Add the repository

Register this repo with Helm and pull the latest chart index.

helm repo add oneuptime https://helm-chart.oneuptime.com/
helm repo update

Available charts

Two charts are published from this repository.

oneuptime

Deploys the full OneUptime platform — monitoring, status pages, incident management, on-call, logs, traces, and metrics.

helm install my-oneuptime \
  oneuptime/oneuptime \
  -f values.yaml

Read the oneuptime chart docs →

kubernetes-agent

Installs the OneUptime Kubernetes monitoring agent into any cluster (standard, GKE Autopilot, EKS Fargate).

helm install my-agent \
  oneuptime/kubernetes-agent \
  -f values.yaml
